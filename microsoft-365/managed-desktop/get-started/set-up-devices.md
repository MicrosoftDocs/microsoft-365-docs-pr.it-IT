# <a name="set-up-microsoft-managed-desktop-devices"></a>Configurare dispositivi Microsoft Managed Desktop

Puoi usare sia i dispositivi nuovi che quelli esistenti in Microsoft Managed Desktop.

## <a name="to-obtain-new-devices"></a>Per ottenere nuovi dispositivi

Ti consigliamo di lavorare con uno dei nostri partner di dispositivi approvati. Puoi collaborare con il contatto dell'account Microsoft per altre informazioni sulla configurazione di una partnership con i dispositivi. In sostanza, tuttavia, il processo è simile al seguente:

1. Esamina l'elenco dei dispositivi attualmente consigliati filtrando i dispositivi Microsoft Managed Desktop nel sito [Shop Windows 10 Pro dispositivi](https://www.microsoft.com/windowsforbusiness/view-all-devices) aziendali.
2. Ordina uno o alcuni esempi dei dispositivi che vuoi usare con un'immagine conforme. L'ordinamento potrebbe [richiedere passaggi di ordinamento specifici.](../service-description/device-images.md)
3. [Convalidare](validate-device.md) i dispositivi di esempio.
5. Dopo la convalida, ordina i dispositivi, collaborando con un partner di dispositivi approvato.
6. Una volta arrivati, eseguire una delle operazioni seguenti:
    - [Registrare manualmente i nuovi dispositivi](register-devices-self.md)
    - Collaborare con un partner per registrare i dispositivi
7. [Prepara gli utenti a](get-started-devices.md) usare i Microsoft Managed Desktop mobili

## <a name="to-reuse-existing-devices"></a>Per riutilizzare i dispositivi esistenti

1. Verifica che i dispositivi esistenti soddisfino i [requisiti del dispositivo](../service-description/device-requirements.md). Puoi anche usare lo [](../get-ready/readiness-assessment-downloadable.md) strumento di verifica della preparazione scaricabile per verificare che un determinato dispositivo soddisfi i requisiti necessari. 
2. Se riutilizzi un dispositivo esistente, potrebbe essere necessario ricrearlo. Per le opzioni dell'immagine, vedi [Immagini dispositivo.](../service-description/device-images.md)
3. Seleziona uno o alcuni esempi dei dispositivi che vuoi riutilizzare e quindi [convalidali.](validate-device.md)
4. Dopo la convalida, eseguire una delle operazioni seguenti:
    - [Registrare manualmente i dispositivi già presenti](register-reused-devices-self.md)
    - Collaborare con un partner per registrare i dispositivi
5. [Prepara gli utenti a](get-started-devices.md) usare i Microsoft Managed Desktop mobili
