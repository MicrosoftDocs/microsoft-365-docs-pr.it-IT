# <a name="enable-user-support-features"></a>Abilitare le funzionalità di supporto degli utenti

Indipendentemente dal fatto che tu supporti il tuo utente o lavori con un partner per fornirlo, segui questi passaggi per consentire al provider di supporto di richiedere l'accesso al dispositivo con privilegi elevati o inoltrare i problemi a Microsoft Managed Desktop, se necessario.

1. Se non ne hanno già uno, gli utenti hanno bisogno di un account nello stesso dominio Azure Active Directory (AAD) dei Microsoft Managed Desktop dispositivi.
2. Aggiungere gli account utente dal passaggio 1 al gruppo di sicurezza **Modern Workplace Roles-Support Partner** in AAD.
