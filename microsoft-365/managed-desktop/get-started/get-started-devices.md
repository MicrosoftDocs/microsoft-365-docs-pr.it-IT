# <a name="get-your-users-ready-to-use-devices"></a>Preparare gli utenti a usare i dispositivi

Una volta che un dispositivo Microsoft Managed Desktop è nelle mani dell'utente, è facile e veloce. I dispositivi vengono preconfigurato con la versione corrente di Windows e le configurazioni e le app vengono installate dal cloud durante il completamento dell'installazione da parte dell'utente. 
 
Per rendere ancora più semplice questa operazione, è disponibile una guida che guida gli utenti nella configurazione iniziale e fornisce risorse della Guida sia per la configurazione che per l'utilizzo in un secondo momento, se necessario. È possibile personalizzare questa guida per includere alcuni dettagli specifici dell'organizzazione. Distribuisci quindi la guida direttamente agli utenti insieme al dispositivo. Esistono due versioni, una per le organizzazioni che usano il sistema di supporto di Microsoft (l'app Guida), una per quelle che forniscono supporto alternativo per gli utenti.

## <a name="to-prepare-the-guide"></a>Per preparare la guida

1. Scaricare la versione appropriata per l'organizzazione:
- [Microsoft Managed Desktop - Introduzione al dispositivo (ottieni](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-help-custom-v3.pdf) la versione della Guida): usa questa versione se gli utenti useranno il sistema di supporto di Microsoft.
- [Microsoft Managed Desktop - Introduzione al dispositivo:](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-custom-v2.pdf) usa questa versione se gli utenti *non useranno* il sistema di supporto di Microsoft.
2. Usa qualsiasi app in grado di aprire file PDF per compilare i dettagli rilevanti per la tua organizzazione:
    - Nome della rete a cui gli utenti devono connettersi per continuare l'installazione (passaggio 3 della guida)
    - Nome dell'account tenant azure dell'organizzazione (passaggio 4 della guida)
    - Informazioni di contatto per il supporto IT interno dell'organizzazione (parte superiore della seconda pagina)
3. Salvare il PDF modificato e quindi distribuirne la distribuzione agli utenti. 

## <a name="ready-to-use-guide"></a>"Guida pronta all'uso"

Viene inoltre disponibile una versione più generica della guida per le organizzazioni che non devono personalizzarla. Esistono due versioni, una per le organizzazioni che usano il sistema di supporto di Microsoft (l'app Guida), una per quelle che forniscono supporto alternativo per gli utenti. 

- [Microsoft Managed Desktop - Introduzione al](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-help-v3.pdf) dispositivo (ottieni la versione della Guida - pronto per l'uso) - usa questa versione se gli utenti useranno il sistema di supporto di Microsoft.
- [Microsoft Managed Desktop - Introduzione al dispositivo (pronto](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-v2.pdf) per l'uso): utilizzare questa versione se gli utenti non utilizzeranno il sistema di supporto di Microsoft.
