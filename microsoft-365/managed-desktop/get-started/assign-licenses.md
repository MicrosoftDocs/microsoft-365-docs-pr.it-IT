# <a name="assign-licenses"></a>Assegnazione delle licenze

Come parte della preparazione per la registrazione a Microsoft Managed Desktop, è necessario assicurarsi di aver ottenuto le licenze necessarie. Se non hai ancora ottenuto le [](../get-ready/prerequisites.md#more-about-licenses) licenze, vedi Altre informazioni sulle licenze per informazioni dettagliate sulle licenze necessarie.


Se le licenze sono tutte allineate, è il momento di assegnarle agli utenti. Per assegnare licenze, è consigliabile sfruttare la funzionalità di gestione delle licenze basata su [gruppo](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-licensing-whatis-azure-portal) di Azure Active Directory.

In caso di difficoltà con l'assegnazione delle licenze, contattare il supporto [dell'amministratore.](../working-with-managed-desktop/admin-support.md)
