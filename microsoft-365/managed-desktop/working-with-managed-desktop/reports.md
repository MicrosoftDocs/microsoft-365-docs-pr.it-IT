# <a name="work-with-reports"></a>Utilizzo dei report

Microsoft Managed Desktop fornisce rapporti che gli amministratori IT del tenant possono utilizzare per comprendere vari aspetti della popolazione dei dispositivi. È possibile accedere ai report nel [portale di amministrazione ](https://aka.ms/mmdportal)in **Reporting**.


## <a name="device-compliance-report"></a>Rapporto di conformità del dispositivo

La visualizzazione principale Mostra la percentuale di dispositivi Microsoft Managed Desktop conformi ai criteri di conformità impostati in Microsoft Intune. È inoltre possibile visualizzare informazioni specifiche per tutti i dispositivi Microsoft Managed Desktop o solo per i dispositivi che non sono conformi ai criteri di conformità ed esportare un file con le stesse informazioni.

Per ulteriori informazioni, vedere [report di conformità del dispositivo](device-compliance-report.md).

## <a name="device-inventory-report"></a>Report inventario dispositivi

La visualizzazione **dispositivi** offre una scheda **Esporta tutto** che consente di [scaricare un report dettagliato sull'inventario](device-inventory-report.md).
