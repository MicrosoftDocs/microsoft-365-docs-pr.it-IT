# <a name="device-inventory-report"></a>Report inventario dei dispositivi

Nella visualizzazione **Dispositivi** puoi selezionare la scheda **Esporta** tutto per scaricare un file delimitato da virgole, incluse queste informazioni:

- Numero di serie
- Nome dispositivo
- Ultimo check-in dispositivo
- Gestito da
- Produttore
- Modello
- Sistema operativo
- Licenza del sistema operativo
- Versione del sistema operativo
- Ownership
- Data di registrazione
- Nome anello
- Archiviazione - Gratuito in GB
- Archiviazione - Totale in GB
- Registrato da UPN utente
- Utente assegnato
- Stato
- Età (mesi)
- Profilo
- Utenti connessi
- Versione TPM
- Avvio protetto abilitato
- Tipo di disco primario
- Memoria fisica totale
