# <a name="device-compliance-report"></a>Rapporto di conformità del dispositivo

Questa visualizzazione Mostra la percentuale di dispositivi Microsoft Managed Desktop conformi ai criteri di conformità impostati in Microsoft Intune.

![Report che mostra la percentuale di dispositivi conformi ai criteri](../../media/mmd-device-compliance-percent.png)


Selezionare **Visualizza dettagli** per visualizzare informazioni specifiche per tutti i dispositivi Microsoft Managed Desktop o solo per i dispositivi che non sono conformi ai criteri di conformità. Utilizzare il menu a discesa per passare da una visualizzazione all'altra. Per essere visualizzato in questa visualizzazione, è necessario che un dispositivo sia un computer fisico e che sia sincronizzato con Intune negli ultimi 30 giorni.

![Riquadro dei dettagli con schede per tutti i dispositivi o dispositivi non conformi. Vengono illustrati lo schema dettagliato che include il numero di serie, il nome del dispositivo, lo stato di conformità e altre informazioni. Scaricare il pulsante dati completi in basso.](../../media/mmd-device-compliance-detail.png)

Da questa visualizzazione, selezionare **Scarica dati completi** per esportare un file delimitato da virgole con le stesse informazioni. Nel file di esportazione le aree visualizzate come **schema** vengono visualizzate come colonne.
