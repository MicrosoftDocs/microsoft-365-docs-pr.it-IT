Per ulteriori informazioni su Microsoft Managed Desktop, vedere [documentazione e risorse di Microsoft Managed Desktop](https://docs.microsoft.com/microsoft-365/managed-desktop/).

Inoltre, il dispositivo deve:     

- Avere un'immagine dalla factory o tramite ri-imaging che:
    - Ha la versione più recente di Windows 10 Pro o Enterprise release precedente o immediata;
    - Ha la versione più recente di Microsoft Office 365 fare clic per eseguire l'esecuzione o la versione precedente immediata.
    - Sono installati i driver più recenti forniti dall'OEM; e
    - Non ha alcun software precaricato aggiuntivo
    - I dispositivi HP con 5VS04AV devono disporre di un'immagine dei dispositivi gestiti Microsoft
- Non è stata completata l'esperienza di Windows per la prima esecuzione.
- Essere registrato con il [pilota automatico di Windows](https://docs.microsoft.com/en-us/windows/deployment/windows-autopilot/user-driven-aad) e assegnare il profilo Autopilot desktop Microsoft gestito.
