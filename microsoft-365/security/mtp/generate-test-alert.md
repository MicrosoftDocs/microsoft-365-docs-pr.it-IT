# <a name="generate-a-test-alert-in-your-microsoft-threat-protection-evaluation-lab"></a>Generare un avviso di testing nel laboratorio di valutazione di Microsoft Threat Protection  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Si applica a:**
- Microsoft Threat Protection

Dopo aver completato l'installazione e la configurazione del Lab di valutazione di Microsoft Threat Protection, è possibile eseguire una simulazione di attacco di minacce per verificare le funzionalità di rilevamento, di correlazione degli avvisi e di correzione automatica della soluzione di protezione integrata.  

1. Accedere a https://security.microsoft.com
2. Nella Home page, cercare una sezione **simulazione**intitolata.  Leggere la guida della procedura dettagliata per la creazione di un test Incident in Microsoft Threat Protection.
