# <a name="microsoft-365-defender-data-security-and-privacy"></a>Protezione e privacy dei dati di Microsoft 365 Defender

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Si applica a:**
- Microsoft 365 Defender

Microsoft 365 Defender opera nei data center di Microsoft Azure nell'Unione europea, nel Regno Unito e negli Stati Uniti. I dati dei clienti raccolti dal servizio sono archiviati a riposo in (a) la posizione geografica del tenant come identificati durante il provisioning o, (b) se Microsoft 365 Defender utilizza un altro servizio Microsoft online per elaborare tali dati, la geolocazione come definito dalle regole di archiviazione dei dati di tale altro servizio online.

I dati dei clienti in formato pseudonimo potrebbero essere archiviati anche nei sistemi di archiviazione e di elaborazione centrali negli Stati Uniti.


Per ulteriori informazioni sulle informazioni relative all'archiviazione dei dati e alla privacy di prodotti specifici, vedere:
- [Microsoft Defender per l'archiviazione e la privacy dei dati di endpoint](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/data-storage-privacy)
- [Sicurezza e privacy dei dati di Microsoft Cloud App Security](https://docs.microsoft.com/cloud-app-security/cas-compliance-trust)
- [Microsoft 365 privacy, sicurezza e trasparenza](https://docs.microsoft.com/office365/servicedescriptions/office-365-platform-service-description/privacy-security-and-transparency#advanced-threat-protection)
