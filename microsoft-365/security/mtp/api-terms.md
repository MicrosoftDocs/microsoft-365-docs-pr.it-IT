## <a name="apis"></a>API

 Le API di Microsoft Threat Protection sono regolate dalla [licenza API Microsoft e dalle condizioni di utilizzo](https://docs.microsoft.com/legal/microsoft-apis/terms-of-use).

### <a name="throttling-limits"></a>Limitazioni

Ogni API ha i propri limiti di limitazione.<br>
I limiti di limitazione sono disponibili per ora e per minuto.<br>
Ad esempio, i limiti di limitazione dell'API degli incidenti sono: 50 chiamate al minuto e 1500 all'ora.<br>
Il codice di stato della risposta HTTP in caso di limitazione sarà 429.
