# <a name="data-monitoring-and-reporting-in-the-microsoft-365-security-center"></a>Monitoraggio dei dati e creazione di report nel Centro sicurezza Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


La categoria **Dati** consente di tenere traccia delle attività degli utenti che potrebbero portare alla divulgazione di dati non autorizzata. Si tratta della rielaborazione dei report dei criteri DLP esistenti e di un rapporto di corrispondenza dei criteri DLP di terze parti.

È possibile visualizzare:

- Utenti che condividono il maggior numero di file dalle app cloud
- Numero di corrispondenze dei criteri DLP
- Il numero di sostituzioni di criteri DLP o falsi positivi riportati
- Numero di corrispondenze ai criteri DLP nei servizi cloud di terze parti tramite Microsoft Cloud App Security
