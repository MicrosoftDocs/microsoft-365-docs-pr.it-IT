# <a name="microsoft-threat-protection"></a>Microsoft Threat Protection

Microsoft Threat Protection è una soluzione di sicurezza completa che unisce potenti funzionalità di sicurezza che consentono alle organizzazioni di oggi di: 

- **Proteggi asset** : le risorse sono protette da utenti, dispositivi e applicazioni non autorizzate.

- **Connect the Dots** -leveraging ineguagliated Optics, la soluzione applica la correlazione dei segnali per raggruppare gli avvisi provenienti da tutte le superfici di attacco per ottenere un contesto migliore nell'ambito di una minaccia.

- **Responsabilizzare i difensori** -fornendo visibilità nell'ambito completo di una minaccia, i difensori della sicurezza sono autorizzati a identificare l'impatto e a prendere le azioni correttive appropriate.


Accedere alla potenza di Microsoft Threat Protection tramite Microsoft 365 Security Center.
