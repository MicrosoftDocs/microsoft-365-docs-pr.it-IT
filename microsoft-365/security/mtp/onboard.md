# <a name="configure-and-manage-microsoft-defender-for-endpoint-capabilities"></a>Configurare e gestire le funzionalità di Microsoft Defender for Endpoint

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

**Si applica a:**

- [Microsoft Defender ATP](https://go.microsoft.com/fwlink/p/?linkid=2069559)

Configura e gestisci tutte le funzionalità di Microsoft Defender for Endpoint per ottenere la migliore protezione della sicurezza per la tua organizzazione. 


## <a name="in-this-section"></a>Contenuto della sezione 
Argomento | Descrizione 
:---|:---
[Configurare le funzionalità di riduzione della superficie di attacco](/windows/security/threat-protection/microsoft-defender-atp/configure-attack-surface-reduction) |  Assicurandosi che le impostazioni di configurazione siano impostate correttamente e che le tecniche di mitigazione degli exploit siano applicate, questo set di funzionalità resiste agli attacchi e allo sfruttamento. 
[Configurare la protezione di nuova generazione](/windows/security/threat-protection/windows-defender-antivirus/configure-windows-defender-antivirus-features) | Configurare la protezione di nuova generazione per rilevare tutti i tipi di minacce emergenti.
[Configurare le funzionalità di Microsoft Threat Experts](/windows/security/threat-protection/microsoft-defender-atp/configure-microsoft-threat-experts) | Configurare e gestire il modo in cui si desidera ottenere l'intelligence sulle minacce per la sicurezza informatica da Microsoft Threat Experts.
[Configurare l'integrazione di Microsoft 365 Defender](/windows/security/threat-protection/microsoft-defender-atp/threat-protection-integration)| Configurare altre soluzioni che si integrano con Microsoft Defender per Endpoint.
[Gestione e supporto api](/windows/security/threat-protection/microsoft-defender-atp/management-apis)| Eseguire il pull degli avvisi nel SIEM o usare le API per creare avvisi personalizzati. Creare e creare report di Power BI. 
[Configurare le impostazioni di Microsoft Defender Security Center](/windows/security/threat-protection/microsoft-defender-atp/preferences-setup) |  Configurare le impostazioni relative al portale, ad esempio le impostazioni generali, le funzionalità avanzate, abilitare l'esperienza di anteprima e altre.
