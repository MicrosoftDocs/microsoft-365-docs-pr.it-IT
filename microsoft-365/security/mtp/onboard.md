# <a name="configure-and-manage-microsoft-defender-for-endpoint-capabilities"></a>Configurare e gestire le funzionalità di Microsoft Defender for Endpoint

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

**Si applica a:**

- [Microsoft Defender ATP](https://go.microsoft.com/fwlink/p/?linkid=2069559)

Configurare e gestire tutte le funzionalità di Microsoft Defender for Endpoint per ottenere la migliore protezione di sicurezza per l'organizzazione. 


## <a name="in-this-section"></a>Contenuto della sezione 
Argomento | Descrizione 
:---|:---
[Configurare le funzionalità di riduzione della superficie di attacco](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-attack-surface-reduction) |  Assicurando che le impostazioni di configurazione siano configurate correttamente e che le tecniche di mitigazione degli exploit siano applicate, questi set di funzionalità resistono agli attacchi e allo sfruttamento. 
[Configurare la protezione di nuova generazione](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/configure-windows-defender-antivirus-features) | Configurare la protezione di nuova generazione per rilevare tutti i tipi di minacce emergenti.
[Configurare le funzionalità di Microsoft Threat Experts](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-microsoft-threat-experts) | Configurare e gestire il modo in cui si desidera ottenere l'intelligence per le minacce alla sicurezza informatica da Microsoft Threat Experts.
[Configurare l'integrazione di Microsoft 365 Defender](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/threat-protection-integration)| Configurare altre soluzioni che si integrano con Microsoft Defender per Endpoint.
[Gestione e supporto api](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/management-apis)| Eseguire il pull degli avvisi nel SIEM o usare le API per creare avvisi personalizzati. Creare e creare report di Power BI. 
[Configurare le impostazioni di Microsoft Defender Security Center](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/preferences-setup) |  Configura le impostazioni correlate al portale, ad esempio le impostazioni generali, le funzionalità avanzate, abilita l'esperienza di anteprima e altre.
