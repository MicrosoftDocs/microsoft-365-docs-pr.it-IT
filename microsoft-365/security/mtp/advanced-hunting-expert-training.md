<span data-ttu-id="e06a3-150">L'estratto seguente di un file CSL mostra un set completo di indicazioni contrassegnate come commenti con `//` .</span><span class="sxs-lookup"><span data-stu-id="e06a3-150">The following excerpt from a CSL file shows a comprehensive set of guidance marked as comments with `//`.</span></span>

```kusto

DeviceLogonEvents
| count

// DeviceLogonEvents
// A table containing a row for each logon a device enrolled in Microsoft Defender for Endpoint
```
