# <a name="use-microsoft-endpoint-manager-and-microsoft-intune-to-configure-and-manage-microsoft-defender-antivirus"></a>Usare Microsoft Endpoint Manager e Microsoft Intune per configurare e gestire Microsoft Defender Antivirus

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Si applica a:**

- [Microsoft Defender per endpoint](/microsoft-365/security/defender-endpoint/)

Se si usa Microsoft Endpoint Manager o Microsoft Intune per gestire gli endpoint nella rete, è ora possibile usare Microsoft Endpoint Manager per gestire le analisi di Microsoft Defender Antivirus.

1. Nell'interfaccia di amministrazione di Microsoft Endpoint Manager ( [https://endpoint.microsoft.com](https://endpoint.microsoft.com) ), accedere a Endpoint **Security**.

2. In **Gestisci** scegliere **Antivirus.**

3. Seleziona i criteri di Microsoft Defender Antivirus. 

4. In **Gestisci** scegliere **Proprietà.**

5. Accanto a **Impostazioni di configurazione** scegliere **Modifica.**

6. Espandere la **sezione Analisi** ed esaminare o modificare le impostazioni di analisi.

7. Scegliere **Revisione e salvataggio**

Hai bisogno di assistenza? Vedi [Gestire la sicurezza degli endpoint in Microsoft Intune.](/mem/intune/protect/endpoint-security)
