# <a name="evaluate-microsoft-defender-for-endpoint"></a>Valutare Microsoft Defender per Endpoint 

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**Si applica a:**
- [Microsoft Defender ATP](https://go.microsoft.com/fwlink/p/?linkid=2154037)
- [Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2118804)

>Vuoi provare Microsoft Defender per Endpoint? [Iscriversi per una versione di valutazione gratuita.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-enablesiem-abovefoldlink)

[Microsoft Defender per endpoint](https://go.microsoft.com/fwlink/?linkid=2154037) è una piattaforma unificata per la protezione preventiva, il rilevamento post-violazione, l'indagine automatizzata e la risposta.

Puoi valutare Microsoft Defender per Endpoint nella tua organizzazione [avviando la versione di valutazione gratuita.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp)

Puoi anche valutare le diverse funzionalità di sicurezza in Microsoft Defender for Endpoint seguendo le istruzioni seguenti.

## <a name="evaluate-attack-surface-reduction"></a>Valutare la riduzione della superficie di attacco

Queste funzionalità consentono di impedire attacchi e sfruttamenti di infettare l'organizzazione.

- [Valutare la riduzione della superficie di attacco](./evaluate-attack-surface-reduction.md)
- [Valutare la protezione da exploit](./evaluate-exploit-protection.md)
- [Valutare la protezione di rete](./evaluate-exploit-protection.md)
- [Valutare l'accesso controllato alle cartelle](./evaluate-controlled-folder-access.md)
- [Valutare Application Guard](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-application-guard/test-scenarios-md-app-guard)
- [Valutare il firewall di rete](https://docs.microsoft.com/windows/security/threat-protection/windows-firewall/evaluating-windows-firewall-with-advanced-security-design-examples)

## <a name="evaluate-next-generation-protection"></a>Valutare la protezione di nuova generazione

Le protezioni di nuova generazione consentono di rilevare e bloccare le minacce più recenti.

- [Valutare l'antivirus](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/evaluate-microsoft-defender-antivirus)
