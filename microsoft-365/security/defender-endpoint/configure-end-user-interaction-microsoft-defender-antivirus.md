# <a name="configure-end-user-interaction-with-microsoft-defender-antivirus"></a>Configurare l'interazione dell'utente finale con Antivirus Microsoft Defender

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Si applica a:**

- [Microsoft Defender per endpoint](/microsoft-365/security/defender-endpoint/)

È possibile configurare il modo in cui gli utenti degli endpoint nella rete possono interagire con Antivirus Microsoft Defender. Puoi configurare se gli utenti visualizzano l'Antivirus Microsoft Defender, quali notifiche visualizzano e se possono ignorare localmente le impostazioni di Criteri di gruppo distribuite a livello globale.

Usa gli articoli seguenti per configurare l'interazione dell'utente finale con Antivirus Microsoft Defender

- [Configurare le notifiche visualizzate sugli endpoint](configure-notifications-microsoft-defender-antivirus.md) Configurare e personalizzare le notifiche, il testo personalizzato per le notifiche e le notifiche sui riavvii per la correzione

- [Impedire agli utenti di visualizzare o interagire con l'Antivirus Microsoft Defender utente](prevent-end-user-interaction-microsoft-defender-antivirus.md) Nascondere l'interfaccia utente agli utenti

- [Impedire agli utenti di modificare localmente le impostazioni dei criteri](configure-local-policy-overrides-microsoft-defender-antivirus.md) Impedire (o consentire) agli utenti di ignorare le impostazioni dei criteri nei singoli endpoint
