# <a name="configure-end-user-interaction-with-microsoft-defender-antivirus"></a>Configurare l'interazione dell'utente finale con Microsoft Defender Antivirus

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Si applica a:**

- [Microsoft Defender per endpoint](/microsoft-365/security/defender-endpoint/)

Puoi configurare il modo in cui gli utenti degli endpoint nella rete possono interagire con Microsoft Defender Antivirus.

Ciò include se visualizzano l'interfaccia di Microsoft Defender Antivirus, quali notifiche visualizzano e se possono ignorare localmente le impostazioni di Criteri di gruppo distribuite a livello globale.
