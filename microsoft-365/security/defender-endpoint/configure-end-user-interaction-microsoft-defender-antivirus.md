# <a name="configure-end-user-interaction-with-microsoft-defender-antivirus"></a>Configurare l'interazione dell'utente finale con Antivirus Microsoft Defender

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Si applica a:**

- [Microsoft Defender per endpoint](/microsoft-365/security/defender-endpoint/)

È possibile configurare il modo in cui gli utenti degli endpoint della rete possono interagire con Antivirus Microsoft Defender. È possibile configurare se gli utenti vedono l'Antivirus Microsoft Defender, quali notifiche vedono e se possono ignorare localmente le impostazioni di Criteri di gruppo distribuite a livello globale.

Utilizzare gli articoli seguenti per configurare l'interazione dell'utente finale con Antivirus Microsoft Defender

- **[Configurare le notifiche visualizzate sugli endpoint](configure-notifications-microsoft-defender-antivirus.md)** Configurare e personalizzare le notifiche, incluso il testo per le notifiche, e le notifiche sui riavvii necessari per la correzione.

- **[Impedire agli utenti di visualizzare o interagire con l'Antivirus Microsoft Defender utente](prevent-end-user-interaction-microsoft-defender-antivirus.md)** Nascondere **l'interfaccia utente & protezione dalle** minacce dei virus agli utenti finali.

- **[Impedire agli utenti di modificare localmente le impostazioni dei criteri](configure-local-policy-overrides-microsoft-defender-antivirus.md)** Impedire (o consentire) agli utenti di ignorare le impostazioni dei criteri nei singoli endpoint.
