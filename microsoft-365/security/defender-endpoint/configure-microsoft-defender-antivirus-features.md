# <a name="configure-microsoft-defender-antivirus-features"></a>Configurare le funzionalità di Microsoft Defender Antivirus

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Si applica a:**

- [Microsoft Defender per endpoint](/microsoft-365/security/defender-endpoint/)

Puoi configurare Microsoft Defender Antivirus con diversi strumenti, tra cui:

- Microsoft Intune
- Microsoft Endpoint Configuration Manager
- Criteri di gruppo
- Cmdlet di PowerShell
- Strumentazione gestione Windows (WMI)

È possibile configurare le seguenti ampie categorie di funzionalità:

- Protezione basata sul cloud
- Protezione sempre in tempo reale, tra cui protezione comportamentale, euristica e basata sull'apprendimento automatico
- Interazione degli utenti finali con il client su singoli endpoint

Gli articoli seguenti descrivono come eseguire le attività principali durante la configurazione di Microsoft Defender Antivirus. Ogni articolo include istruzioni per lo strumento o gli strumenti di configurazione applicabili.

|Articolo  |Descrizione  |
|---------|---------|
|[Utilizzare la protezione Microsoft Defender Antivirus fornita dal cloud Microsoft](cloud-protection-microsoft-defender-antivirus.md)     | Usa la protezione basata sul cloud per un rilevamento antivirus avanzato, veloce e affidabile.        |
|[Configurare la protezione comportamentale, euristica e in tempo reale](configure-protection-features-microsoft-defender-antivirus.md)     |Abilitare la protezione antivirus basata sul comportamento, euristica e in tempo reale.         |
|[Configurare l'interazione dell'utente finale con Microsoft Defender Antivirus](configure-end-user-interaction-microsoft-defender-antivirus.md) | Configura il modo in cui gli utenti finali nell'organizzazione interagiscono con Microsoft Defender Antivirus, quali notifiche visualizzano e se possono ignorare le impostazioni. |

> [!TIP]
> È inoltre possibile consultare gli argomenti [di riferimento](configuration-management-reference-microsoft-defender-antivirus.md) per gli strumenti di gestione e configurazione per una panoramica di ogni strumento e collegamenti a ulteriori informazioni.
