# <a name="send-and-receive-smime-signed-and-encrypted-email-in-exchange-online"></a>Inviare e ricevere posta elettronica firmata e crittografata S/MIME in Exchange Online

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]


La procedura di invio di un messaggio crittografato con S/MIME in Microsoft Outlook è molto simile a quella di invio di un messaggio non crittografato. Per ulteriori informazioni sulla lettura o l'invio di messaggi crittografati con S/MIME da un programma di posta elettronica come Outlook sul Web (in precedenza noto come Outlook Web App), vedere [Encrypt messages by using S/MIME in Outlook on the web](https://support.microsoft.com/office/878c79fc-7088-4b39-966f-14512658f480).
