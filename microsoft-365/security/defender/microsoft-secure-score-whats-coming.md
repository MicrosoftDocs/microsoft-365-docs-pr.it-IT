# <a name="whats-coming-to-microsoft-secure-score"></a>Novità di Microsoft Secure Score

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]

Microsoft Secure Score è disponibile nel centro sicurezza https://security.microsoft.com/securescore [Microsoft 365 sicurezza.](overview-security-center.md)

## <a name="proposed-changes"></a>Modifiche proposte

Stiamo apportando alcune modifiche nel prossimo futuro per rendere [Microsoft Secure Score](microsoft-secure-score.md) un migliore rappresentante della tua posizione di sicurezza e migliorare l'usabilità. Il punteggio e il punteggio massimo possibile possono cambiare.

### <a name="july-2021"></a>Luglio 2021

#### <a name="add-improvement-action-related-to-microsoft-teams"></a>Aggiungere azioni di miglioramento correlate a Microsoft Teams

- Impedire agli utenti con accesso remoto di ignorare una sala di attesa per le riunioni.
- Limitare ai partecipanti esterni di avere il controllo in una Teams riunione.
- Impedire agli utenti anonimi di avviare Teams riunioni.
- Richiedere che le lobby siano impostate per Teams riunioni.
- Configurare quali utenti possono essere presenti nelle Teams riunioni.

#### <a name="add-improvement-action-related-to-microsoft-defender-for-endpoint"></a>Aggiungere un'azione di miglioramento correlata a Microsoft Defender per Endpoint
- Risolvere i problemi relativi alla raccolta dei dati dei sensori di Microsoft Defender for Endpoint per macOS
- Risolvere i problemi di comunicazione di Microsoft Defender for Endpoint per macOS
- Impostare la lunghezza minima della password su 15 o più caratteri in macOS
- Imposta "Imponi cronologia password" su "24 o più password" in macOS
- Impostare "Validità massima password" su "90 o meno giorni, ma non su 0" in macOS
- Impostare la soglia di blocco dell'account su 5 o su un valore inferiore in macOS
- Attivare Firewall su macOS
- Abilita Gatekeeper
- Abilitazione della protezione dell'integrità del sistema (SIP)
- Abilitare Crittografia disco FileVault
- Impostare lo schermo in modo che si blocchi all'avvio di screensaver in macOS
- Verificare che screensaver sia impostato per l'avvio tra 20 minuti o meno in macOS
- Proteggere le cartelle home
- Attivare la Antivirus Microsoft Defender in tempo reale per macOS
- Attivare la Antivirus Microsoft Defender protezione puA in modalità blocco per macOS
- Abilitare Antivirus Microsoft Defender di protezione cloud per macOS
- Aggiornare Antivirus Microsoft Defender definizioni per macOS
- Risolvere i problemi relativi alla raccolta dei dati dei sensori di Microsoft Defender for Endpoint per Linux
- Risolvere i problemi di comunicazione di Microsoft Defender per Endpoint per Linux
- Account di accesso senza restrizioni
- Attivare la Antivirus Microsoft Defender in tempo reale per Linux
- Attivare la Antivirus Microsoft Defender puA in modalità blocco per Linux
- Abilitare Antivirus Microsoft Defender protezione basata sul cloud per Linux
- Aggiornare Antivirus Microsoft Defender definizioni per Linux
