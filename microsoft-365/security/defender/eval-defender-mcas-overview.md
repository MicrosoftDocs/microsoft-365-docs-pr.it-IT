# <a name="evaluate-microsoft-cloud-app-security"></a>Valutare Microsoft Cloud App Security

**Si applica a:**
- Microsoft 365 Defender


In questo articolo viene descritto il processo per abilitare e pilotare Microsoft Cloud App Security insieme Microsoft 365 Defender. Prima di avviare questo processo, verificare di aver [](eval-overview.md) esaminato il processo complessivo per la valutazione Microsoft 365 Defender e di aver creato l'Microsoft 365 Defender [di valutazione.](eval-create-eval-environment.md) 
<br>

Usa la procedura seguente per abilitare e pilotare la sicurezza delle app cloud Microsoft.

![Passaggi per l'aggiunta di Microsoft Defender per Office all'ambiente di valutazione di Defender](../../media/defender/m365-defender-office-eval-steps.png)



|Passaggio  |Descrizione  |
|---------|---------|
|[Esaminare i requisiti dell'architettura e i concetti chiave](eval-defender-mcas-architecture.md)    | Comprendere l'Cloud App Security e come si integra con Microsoft 365 Defender, Microsoft Defender for Endpoint e Azure Active Directory.        |
|[Abilitare l'ambiente di valutazione](eval-defender-mcas-enable-eval.md)     | Connessione al portale, configurare l'integrazione con Defender per l'identità e/o i dispositivi di rete dell'organizzazione e iniziare a visualizzare e gestire le app cloud.         |
|[Configurare il progetto pilota ](eval-defender-mcas-pilot.md)    | Impostare l'ambito della distribuzione su determinati gruppi di utenti, configurare il controllo dell'app di accesso condizionale e provare le esercitazioni per la protezione dell'ambiente.       |
