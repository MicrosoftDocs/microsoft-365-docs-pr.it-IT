# <a name="evaluate-microsoft-defender-for-endpoint-overview"></a>Panoramica della valutazione di Microsoft Defender for Endpoint

**Si applica a:**

- Microsoft 365 Defender


Questo articolo descrive il processo per abilitare e pilotare Microsoft Defender for Endpoint. Prima di avviare questo processo, verificare di aver [](eval-overview.md) esaminato il processo complessivo per la valutazione Microsoft 365 Defender e di aver creato l'Microsoft 365 Defender [di valutazione.](eval-create-eval-environment.md) 
<br>

Usa la procedura seguente per abilitare e pilotare Microsoft Defender per Endpoint.

![Passaggi per l'aggiunta di Microsoft Defender per Endpoint all'ambiente di valutazione di Defender](../../media/defender/m365-defender-endpoint-eval-steps.png)


Nella tabella seguente vengono descritti i passaggi illustrati nell'illustrazione.

 |Passaggio   |Descrizione
|---------|---------|
| [Passaggio 1. Esaminare i requisiti dell'architettura e i concetti chiave](eval-defender-endpoint-architecture.md)    | Comprendere l'architettura di Defender for Endpoint e le funzionalità disponibili.       |
|[Passaggio 2. Abilitare l'ambiente di valutazione](eval-defender-office-365-enable-eval.md)     |   Seguire i passaggi per configurare l'ambiente di valutazione.      |
|[Passaggio 3. Configurare il progetto pilota ](eval-defender-office-365-pilot.md)    |    Verificare il gruppo pilota, eseguire simulazioni e acquisire familiarità con le funzionalità e i dashboard chiave.     |
