# <a name="move-files-to-sharepoint"></a>Spostare i file in SharePoint

> [!VIDEO https://www.microsoft.com/videoplayer/embed/65ec460e-296c-458f-b004-906756dfb3d0?autoplay=false]

Dopo aver eseguito l'iscrizione a Microsoft 365 Business Premium, è&#39;necessario copiare i file di lavoro della società in SharePoint. SharePoint è un buon posto in cui archiviare i file aziendali condivisi a cui tutti devono accedere. In genere sostituisce l'utilizzo di una condivisione di file o di un'unità di rete.

## <a name="try-it"></a>Perché non provarlo?

1. Aprire Microsoft teams e quindi aprire un team a cui tutti gli utenti dell'azienda hanno accesso.
2. Selezionare  **i file** e quindi selezionare  **Apri in SharePoint**.
3. Selezionare  **documenti** per passare al percorso in cui è archiviato tutto il sito.
4. Selezionare  **Sincronizza** , quindi fare clic su  **Sì**. In questo modo i file del sito di SharePoint vengono sincronizzati nel computer in uso.
5. In Esplora file è&#39;possibile visualizzare la cartella documenti sincronizzati dal sito di SharePoint della società&#39;. Se si sincronizzano altri siti di SharePoint, verranno visualizzati al di sopra o al di sotto di questo. Fare clic con il pulsante destro del mouse sulla cartella documenti sincronizzati e aprirla in una nuova finestra per visualizzarla fianco a fianco con i file aziendali che si desidera spostare.
6. Selezionare tutti i file che si desidera spostare e trascinarli nella cartella di SharePoint sincronizzata. Le frecce blu circolari accanto ai file indicano che&#39;ri sincronizzazione nel sito di SharePoint. Quando si&#39;il completamento della sincronizzazione, vengono visualizzati i segni di spunta verdi.
7. Tornare al sito di SharePoint per visualizzare i file che sono stati sincronizzati con il cloud.

Oltre a lavorare sui file direttamente dal computer, è possibile accedervi in un Web browser su qualsiasi computer.

È ora possibile accedere ai file dal sito o dai team di SharePoint.
