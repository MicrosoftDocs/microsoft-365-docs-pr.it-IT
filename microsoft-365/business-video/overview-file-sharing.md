# <a name="overview-of-sharing-business-files"></a>Panoramica della condivisione di file aziendali

È necessario condividere i file con client, clienti, partner, fornitori e altre persone dell'azienda. Come si condividono i file in modo che tutti possano trovarli, aprirli e modificarli?

|![Condivisione sicura](../media/securely-share-file.png)<br/>[Condivisione di un file con un utente esterno alla propria azienda](#share-a-file-with-someone-outside-of-your-company)|![Collaborare con un client](../media/share-and-collab-with-partner.png) <br/>[Condividere e collaborare con un client o un partner aziendale](#share-and-collaborate-with-a-client-or-business-partner) | ![Condivisione all'interno dell'organizzazione](../media/share-inside-your-org.png) <br/>[Condivisione all'interno dell'azienda](#share-inside-your-business) |

## <a name="share-a-file-with-someone-outside-of-your-company"></a>Condivisione di un file con un utente esterno alla propria azienda

- Condividere un file con un utente esterno alla propria azienda (ad esempio un client o un cliente) tramite posta elettronica se necessario solo per andare in un modo.
- Se si ha bisogno di una risposta o si desidera che compilano alcune informazioni e lo rimandino, è possibile utilizzare SharePoint o Microsoft teams per condividerlo con un collegamento di modifica.

Ulteriori informazioni su come [condividere i file all'esterno dell'organizzazione](securely-share-files-externally.md) con collegamenti sicuri e su come [condividere i file con un collegamento a un utente](share-files-externally.md).

## <a name="share-and-collaborate-with-a-client-or-business-partner"></a>Condividere e collaborare con un client o un partner aziendale

Condivisione come parte di un progetto? O come parte di una relazione continua con un partner aziendale o un fornitore? È possibile inserire solo un po' di tempo e configurare un team in Microsoft teams e avere un posto in cui condividere e collaborare. È facile collaborare con utenti esterni alla propria azienda ed è anche possibile impostare canali separati per fornitori o partner aggiuntivi per mantenere tutto in chiaro.

1. Creare il team.
1. Caricare i file.
1. Invitare i propri partner o fornitori al team come utenti "Guest" per fornire loro l'accesso.

Ulteriori informazioni su come [condividere i file all'esterno dell'organizzazione con il collegamento sicuro](https://support.microsoft.com/office/7266f44e-3e06-4736-b9d3-0580c24bba34)s.

## <a name="share-inside-your-business"></a>Condivisione all'interno dell'azienda

Se si lavora con i file che si condividono all'interno dell'azienda, è anche consigliabile configurare un team in Microsoft teams. Condividere i file presenti e tutti gli utenti dell'azienda possono trovarli e modificarli facilmente e sapere che tutti gli utenti stanno lavorando sulla stessa pagina.

1. Creare il team e invitare il team.
1. Caricare i file.

Ulteriori informazioni sull'archiviazione dei [file dell'azienda](https://support.microsoft.com/office/e4d98e10-3532-4eed-85d6-92728454e32b) e sull' [archiviazione di file dipendente](https://support.microsoft.com/office/12dbe3e4-dbef-48f8-a90e-87f1bc607073).
