# <a name="reset-passwords-for-your-users"></a>Reimpostare le password per gli utenti

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FVVP?autoplay=false]

In Microsoft 365 per le aziende, è possibile reimpostare le password per gli utenti. È anche possibile consentire ai dipendenti di reimpostare le proprie password. Per [ulteriori informazioni, vedere Set up self-service password reset.](set-up-self-serve-password-reset.md)

## <a name="try-it"></a>Perché non provarlo?

1. Quando un utente richiede una nuova password, si riceverà una richiesta di reimpostazione della password tramite posta elettronica. Per reimpostare la password, apri l'icona di avvio delle app e seleziona **Amministratore.**
1. Nell'interfaccia di amministrazione di Microsoft 365 selezionare Utenti **,** Utenti  **attivi** e quindi selezionare l'icona della chiave accanto all'utente che ha richiesto la reimpostazione.
1. Selezionare **Genera automaticamente password** per creare automaticamente una password casuale.
1. Selezionare **Reimposta.**
1. Selezionare la **casella di controllo Invia password tramite posta** elettronica.
1. You'll see your email address in the **Email the new password to the following recipients** box. Aggiungere l'indirizzo di posta elettronica alternativo dell'utente o qualsiasi indirizzo scelto.
1. Scegliere **Invia messaggio di posta elettronica e chiudere**.

    L'utente riceverà un messaggio di posta elettronica con le istruzioni per la reimpostazione della password.
