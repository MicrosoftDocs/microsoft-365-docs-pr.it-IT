# <a name="where-to-store-files"></a>Dove archiviare i file

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FTHX?autoplay=false]

In Microsoft 365, è possibile archiviare il lavoro in OneDrive for Business o SharePoint siti. Ogni persona dell'organizzazione dispone di una raccolta OneDrive for Business per archiviare i file di lavoro personali. Quando crei e salvi i file in una cartella OneDrive, questi vengono sincronizzati con il cloud in modo da poterli accedere da qualsiasi luogo, ovvero un web browser, un computer o un dispositivo mobile. Per l'archiviazione di file condivisi e la collaborazione in team, usare un SharePoint locale. SharePoint anche i siti vengono sincronizzati con il cloud per un facile accesso da qualsiasi luogo.

## <a name="try-it"></a>Perché non provarlo?

### <a name="store-or-share-files-with-onedrive"></a>Archiviare o condividere file con OneDrive

1. Nel computer usare Esplora file per aprire OneDrive. In caso contrario, [Office 365](https://www.office.com/)aprire **OneDrive'icona** di avvio delle app.
2. Visualizzare e caricare file personali oppure condividere documenti o cartelle facendo clic su  **Condividi**  e quindi invitando altri utenti a visualizzare i documenti o inviando loro un collegamento. [Altre informazioni](https://support.microsoft.com/office/9fcc2f7d-de0c-4cec-93b0-a82024800c07#os_type=onedrive_-_business).

### <a name="collaborate-with-a-sharepoint-site"></a>Collaborare con un SharePoint sito

1. Da [Office 365](https://www.office.com/), **aprire** SharePoint dall'icona di avvio delle app e quindi selezionare il sito del team.
2. Aggiungere file al percorso predefinito Documenti o creare più percorsi di documenti, denominati Raccolte.

È possibile personalizzare il SharePoint creando siti secondari per progetti, team o clienti specifici. È inoltre possibile aggiungere il nome della società, il logo e l'aspetto, trasformando il sito in una intranet aziendale. [Altre informazioni](https://support.microsoft.com/office/06bbadc3-6b04-4a60-9d14-894f6a170818).
