# <a name="where-to-store-files"></a>Dove archiviare i file

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE1FTHX?autoplay=false]

In Microsoft 365, è possibile archiviare i propri lavori in OneDrive for business o nei siti di SharePoint. Ogni persona all'interno dell'organizzazione dispone di una propria libreria di OneDrive for business per archiviare i file di lavoro personali. Quando si creano e si salvano file in una cartella di OneDrive, vengono sincronizzati con il cloud in modo che sia possibile accedervi da qualsiasi luogo, ovvero un Web browser, un computer o un dispositivo mobile. Per l'archiviazione dei file condivisi e la collaborazione del team, utilizzare un sito di SharePoint. Anche i siti di SharePoint vengono sincronizzati con il cloud per un facile accesso da qualsiasi luogo.

## <a name="try-it"></a>Perché non provarlo?

### <a name="store-or-share-files-with-onedrive"></a>Archiviare o condividere file con OneDrive

1. Nel computer usare Esplora file per aprire OneDrive. In alternativa, da [Office 365](https://www.office.com/)aprire  **OneDrive**  dall'icona di avvio delle app.
2. Consente di visualizzare e caricare i file personali oppure di condividere documenti o cartelle facendo clic su  **Condividi**  e quindi invitare altri utenti a visualizzare i documenti o a inviargli un collegamento. [Altre informazioni](https://support.microsoft.com/office/9fcc2f7d-de0c-4cec-93b0-a82024800c07#os_type=onedrive_-_business).

### <a name="collaborate-with-a-sharepoint-site"></a>Collaborare con un sito di SharePoint

1. Da [Office 365](https://www.office.com/), aprire  **SharePoint**  dall'icona di avvio delle app e quindi selezionare il sito del team.
2. Aggiungere file al percorso dei documenti predefiniti oppure creare altre posizioni del documento, denominate raccolte.

È possibile personalizzare il sito di SharePoint creando siti secondari per progetti, team o clienti specifici. È inoltre possibile aggiungere il nome della società, il logo e l'aspetto, trasformando il sito in una rete Intranet aziendale. [Altre informazioni](https://support.microsoft.com/office/06bbadc3-6b04-4a60-9d14-894f6a170818).
