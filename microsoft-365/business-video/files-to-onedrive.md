# <a name="move-files-to-onedrive"></a>Spostare i file in OneDrive

## <a name="watch-move-your-files-to-onedrive-for-business"></a>Watch: Move your files to OneDrive for Business

> [!VIDEO https://www.microsoft.com/videoplayer/embed/d74b083c-1f44-43ea-8a14-2e1fc600b341?autoplay=false]

Dopo l'iscrizione Microsoft 365 Business Premium, è&#39;copiare i file di lavoro esistenti in OneDrive for Business. OneDrive è un buon punto in cui archiviare i file di lavoro personali o i file aziendali sensibili a cui si desidera controllare l'accesso.

## <a name="try-it"></a>Perché non provarlo?

1. Da Windows, selezionare il **pulsante Start.**
2. Cercare OneDrive e selezionarlo.
3. Immettere l'indirizzo di posta elettronica dell'attività.
4. Selezionare **Accedi** e quindi **Avanti.**
5. Passare alla breve esercitazione e quindi selezionare Apri **la cartella OneDrive.** Se usi anche la versione personale di OneDrive,&#39;vedrai qui.
6. In Esplora file passare ai file che si desidera copiare.
7. Seleziona i file e quindi trascinali nella cartella OneDrive cartella. Le frecce circolari blu accanto ai file significano che&#39;la sincronizzazione con il cloud. Al termine&#39;la sincronizzazione, vengono visualizzati segni di spunta verdi.

Oltre a lavorare sui file OneDrive locale, è possibile accedervi dal Web browser:

1. Nel Web browser, accedere a office.com posta elettronica di lavoro.
2. Scegliere **OneDrive**. Tutti i file sono elencati e puoi lavorarvi nel browser da qualsiasi computer.
