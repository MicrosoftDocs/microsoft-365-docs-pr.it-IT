# <a name="move-files-to-onedrive"></a>Spostare i file in OneDrive

> [!VIDEO https://www.microsoft.com/videoplayer/embed/d74b083c-1f44-43ea-8a14-2e1fc600b341?autoplay=false]

Dopo aver eseguito l'iscrizione a Microsoft 365 Business Premium, è&#39;necessario copiare i file di lavoro esistenti in OneDrive for business. OneDrive è un ottimo posto in cui archiviare i file di lavoro personali o i file aziendali riservati a cui si desidera controllare l'accesso.

## <a name="try-it"></a>Perché non provarlo?

1. Da Windows, selezionare il pulsante  **Start** .
2. Cercare OneDrive e selezionarlo.
3. Immettere l'indirizzo di posta elettronica di lavoro.
4. Selezionare  **Accedi** e quindi fare clic su  **Avanti**.
5. Passare attraverso l'esercitazione breve, quindi selezionare  **Apri la cartella OneDrive**. Se si utilizza anche la versione personale di OneDrive, è&#39;visualizzarla qui.
6. In Esplora file passare ai file che si desidera copiare.
7. Selezionare i file, quindi trascinarli e rilasciarli nella cartella OneDrive. Le frecce blu circolari accanto ai file indicano che&#39;ri sincronizzazione nel cloud. Quando si&#39;il completamento della sincronizzazione, vengono visualizzati i segni di spunta verdi.

Oltre a lavorare sui file di OneDrive localmente, è possibile accedervi dal Web browser:

1. Nel browser Web, accedere a office.com con il proprio indirizzo di posta elettronica di lavoro.
2. Scegliere  **OneDrive**. Tutti i file sono elencati ed è possibile utilizzarli nel browser da qualsiasi computer.
