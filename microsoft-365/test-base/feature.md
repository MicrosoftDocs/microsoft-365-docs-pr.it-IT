# <a name="windows-feature-update-validation"></a>Windows Convalida dell'aggiornamento delle funzionalità

Sono necessarie informazioni dettagliate sulle prestazioni delle applicazioni con la prossima versione di Windows 10 o Windows 11, senza mantenere un ambiente per convalidare le nuove funzionalità Windows? 

Si desidera eseguire i test di convalida Windows build del programma Insider nell'ambiente Azure?

**La convalida degli** aggiornamenti delle funzionalità in Test Base per M365 può aiutarti a ottenere tutti questi e molto altro ancora!

Per informazioni su come accedere a questa nuova funzionalità, vedere la struttura dettagliata riportata di seguito in Test Base per il servizio M365.

Per iniziare in Test Base per M365, caricare le applicazioni (e i file correlati) tramite il portale di ```Feature update validation``` onboarding self-service. 

Di seguito sono evidenziati i passaggi da eseguire durante la compilazione dei **dettagli del test:**

1. Selezionare **Aggiornamento funzionalità come** tipo di aggiornamento del sistema operativo:

![Tipo di sistema operativo di convalida dell'aggiornamento delle funzionalità](Media/Feature-update-validation-01.png)

2. Scegli il Windows Insider Channel in base al quale vuoi convalidare l'applicazione.  

![Convalida dell'aggiornamento delle funzionalità. Scelta del canale beta Insider](Media/Feature-update-validation-02.png)

3. Seleziona una versione sul mercato di Windows 10 o Windows 11 come base per il test (e le informazioni dettagliate risultanti)e fornisci gli altri dettagli necessari per eseguire correttamente l'onboardboard del pacchetto.

![Convalida dell'aggiornamento delle funzionalità con versioni rilasciate di Windows 10 e Windows 11](Media/Feature-update-validation-03.png)

4. Per visualizzare i risultati della convalida dell'applicazione rispetto agli aggiornamenti Windows 10 funzionalità non rilasciati, visitare il ```Feature Updates Test Results``` sito Web .

![La convalida degli aggiornamenti delle funzionalità consente di esaminare rapidamente i risultati](Media/Feature-update-validation-04.png)
