# <a name="mobility--people-experiences"></a>Mobilità - Esperienze degli utenti

Il punteggio di produttività fornisce informazioni dettagliate sul percorso di trasformazione digitale dell'organizzazione attraverso l'uso di Microsoft 365 e le esperienze tecnologiche che lo supportano. Il punteggio dell'organizzazione riflette le misurazioni delle persone e dell'esperienza tecnologica e può essere confrontato con i benchmark di organizzazioni simili al tuo. La categoria di mobilità fa parte delle misure dell'esperienza utente. Per altre informazioni, consultare la panoramica [del punteggio di produttività](productivity-score.md) e leggere [l'Informativa sulla privacy di Microsoft.](https://privacy.microsoft.com/privacystatement)

## <a name="prerequisites"></a>Prerequisiti

Per iniziare a usare Dati statistici per dispositivi mobili, gli utenti dell'organizzazione devono disporre di una licenza per:

- Microsoft Teams
- Exchange Online
- Word
- Excel
- PowerPoint
- OneNote

Per ulteriori informazioni, vedere [assegnare licenze agli utenti.](../manage/assign-licenses-to-users.md)

Dopo che le persone sono state attive nei prodotti di cui sopra almeno una volta negli ultimi 28 giorni, inizierà a vedere le informazioni dettagliate.

## <a name="why-your-organization39s-mobility-score-matters"></a>Perché l'organizzazione&#39;punteggio di mobilità è importante

Un pilastro fondamentale della produttività dell'organizzazione è la capacità delle persone di lavorare in modo flessibile ovunque ci si trova. Con Microsoft 365, le persone possono rimanere in contatto con Outlook, Microsoft Teams e Yammer. Gli utenti possono inoltre collaborare facilmente ai contenuti utilizzando Word, Excel, PowerPoint e OneNote da qualsiasi posizione e piattaforma.

## <a name="how-we-calculate-the-score"></a>Come calcoliamo il punteggio

Forniamo una panoramica principale dell'esperienza che contiene le metriche chiave per questa categoria. Viene quindi usato un framework di punteggio dettagliato di seguito per queste metriche per calcolare il punteggio dell'organizzazione.

### <a name="primary-insight"></a>Informazioni principali

Microsoft 365 consente agli utenti di lavorare in modo flessibile tra le app, tra cui Microsoft Outlook, Word, Excel, PowerPoint, OneNote, Microsoft Teams, Yammer e Skype for Business. Gli utenti possono inoltre lavorare ovunque si trovino utilizzando un'esperienza semplice su piattaforme desktop, Web e per dispositivi mobili. Le informazioni principali esaminano i prodotti abilitati per gli utenti dell'organizzazione e quante di queste persone sono attive su almeno due piattaforme.

:::image type="content" source="../../media/primary-mobility.png" alt-text="Grafico che mostra le informazioni principali per la mobilità: il punteggio di produttività delle esperienze degli utenti.":::

1. **Intestazione:** Mostra la percentuale di persone nell'organizzazione che hanno accesso a una qualsiasi delle app di Microsoft 365 e che sono attive in almeno una di queste applicazioni su più piattaforme.
2. **Corpo:** Fornisce ulteriori informazioni su come l'uso di queste applicazioni su più piattaforme può promuovere efficienza e soddisfazione.
3. **Visualizzazione (stato corrente):** Mostra quante persone usano più piattaforme su desktop, dispositivi mobili e Web per almeno un Microsoft 365 (elenco di seguito) come segue:
    - **Barra orizzontale** in cui la parte blu (colorata) rappresenta la percentuale espressa nell'intestazione.
    - **La frazione** sulla barra evidenzia il numeratore/denominatore utilizzato per calcolare la percentuale nell'intestazione.
        - Numeratore: il numero di persone nell'organizzazione che usano qualsiasi applicazione in Microsoft Outlook, Word, Excel, PowerPoint, OneNote e Microsoft Teams, Yammer e Skype su più piattaforme da desktop, Web e dispositivi mobili negli ultimi 28 giorni.
        - Denominatore: il numero di persone con licenza per Microsoft 365 Apps, Exchange, Yammer, Microsoft Teams o Skype per almeno 1 degli ultimi 28 giorni.
    - Anche **il valore di benchmark** peer per la metrica chiave viene visualizzato come percentuale.
4. **Visualizzare le risorse per dispositivi mobili:** Selezionare questo collegamento per visualizzare il contenuto della Guida.

#### <a name="trend-visualization-of-the-primary-insight"></a>Visualizzazione delle tendenze delle informazioni principali

Questo grafico mostra la linea di tendenza, dove il numeratore è il numero di persone che hanno usato le app negli ultimi 180 giorni. Ogni punto dati nel grafico a linee è un aggregato di attività degli ultimi 28 giorni. Ogni punto dati fornisce un conteggio di tutte le persone nell'organizzazione che usano un'applicazione su almeno due piattaforme negli ultimi 28 giorni per ogni data sull'asse x.

### <a name="scoring-framework"></a>Framework di punteggio

Il punteggio di mobilità per l'organizzazione misura a livello di organizzazione (aggregato) se le persone usano Microsoft 365 Apps - Outlook, Teams, Word, Excel, PowerPoint, OneNote, Yammer e Skype - tra le diverse piattaforme: desktop, Web e dispositivi mobili.

I punteggi non vengono forniti a livello di singolo utente.

## <a name="explore-how-your-org-works-across-platforms-and-locations"></a>Esplorare il funzionamento dell'organizzazione su piattaforme e posizioni diverse

Queste metriche aggiuntive non contribuiscono direttamente al punteggio di produttività, ma consentono di creare un piano d'azione come parte della trasformazione digitale.

2. **Corpo:** Vengono fornite informazioni sul valore dell'utilizzo di Outlook su dispositivi mobili per rimanere connessi ovunque si trovino messaggi di posta elettronica.
3. **Visualizzazione:** Mostra la percentuale di persone attive in Outlook e che utilizzano una o più piattaforme:
      - **Più piattaforme:**
        - Numeratore: il numero di persone che hanno utilizzato Outlook su almeno due piattaforme da desktop, mobile o Web negli ultimi 28 giorni.
        - Denominatore: il numero di persone che hanno utilizzato Outlook almeno una volta negli ultimi 28 giorni.
      - **Solo desktop:**
        - Numeratore: il numero di persone che hanno utilizzato Outlook solo su una piattaforma desktop negli ultimi 28 giorni.
        - Denominatore: il numero di persone che hanno utilizzato Outlook almeno una volta negli ultimi 28 giorni
      - **Solo Web:**
        - Numeratore: il numero di persone che hanno utilizzato Outlook solo su una piattaforma Web negli ultimi 28 giorni.
        - Denominatore: il numero di persone che hanno utilizzato Outlook almeno una volta negli ultimi 28 giorni.
      - **Solo dispositivi mobili:**
        - Numeratore: numero di persone che hanno utilizzato Outlook solo su una piattaforma mobile negli ultimi 28 giorni.
        - *Denominatore: numero di persone che hanno utilizzato Outlook almeno una volta negli ultimi 28 giorni.
