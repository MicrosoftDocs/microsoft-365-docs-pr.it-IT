# <a name="manage-device-access-settings-in-basic-mobility-and-security"></a>Gestire le impostazioni di accesso ai dispositivi in Sicurezza e mobilità di base

Se si usa Basic Mobility and Security, potrebbero essere presenti dispositivi che non è possibile gestire con Basic Mobility and Security. In tal caso, è consigliabile bloccare Exchange ActiveSync'accesso dell'app alla posta elettronica di Microsoft 365 per i dispositivi mobili non supportati da Basic Mobility and Security. Ciò consente di proteggere le informazioni dell'organizzazione su più dispositivi.

Eseguire la procedura seguente:

1. Accedere a Microsoft 365 con l'account di amministratore globale.

2. Nel browser digitare:  [https://protection.office.com](https://protection.office.com/) .    

    >[!IMPORTANT]
    >Se questa è la prima volta che si usa Basic Mobility and Security per Microsoft 365 Business Standard, attivarlo qui: Attivare sicurezza e [mobilità di base.](https://admin.microsoft.com/EAdmin/Device/IntuneInventory.aspx) Dopo l'attivazione, gestire i dispositivi con [Office 365 Security & Compliance.](https://protection.office.com/)

3. Passare a Prevenzione della perdita dei dati > **criteri** dispositivo di gestione dei dispositivi e selezionare Gestisci impostazioni di accesso ai dispositivi a livello di   >  **** **organizzazione.**

4. Selezionare **Blocca.**

    :::image type="content" source="../../media/basic-mobility-security/bms-5-block-access.png" alt-text="Casella di controllo Accesso di base per dispositivi mobili e sicurezza":::

5. Selezionare **Salva.**

Per informazioni sui dispositivi supportati da Dispositivi mobili e sicurezza di base, vedere [Funzionalità di sicurezza e mobilità di base.](capabilities.md)
