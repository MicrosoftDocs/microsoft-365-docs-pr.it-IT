# <a name="download-perpetual-software-and-product-license-keys"></a>Scaricare il software perpetuo e i codici product license

Questo articolo spiega come scaricare i codici di licenza software e di prodotto per il software perpetuo acquistato tramite il programma Cloud Solution Provider (CSP).

## <a name="before-you-begin"></a>Prima di iniziare

Per eseguire la procedura descritta in questo articolo, è necessario essere un amministratore globale. Per altre informazioni, vedere [Informazioni sui ruoli di amministratore](../add-users/about-admin-roles.md).

## <a name="download-software-and-product-license-keys"></a>Scaricare codici product license e software

1. Nell'interfaccia di amministrazione di Microsoft 365, passare alla pagina **Fatturazione** > <a href="https://go.microsoft.com/fwlink/p/?linkid=842054" target="_blank">I tuoi prodotti</a>.
2. Nella **sezione** **Software** della scheda Prodotti selezionare il software che si desidera scaricare.
3. Nella sezione Download dei  codici & della pagina dei dettagli dell'abbonamento scegliere il tipo di file Versione prodotto , Lingua e & **CPU** e quindi selezionare **Scarica**.  
4. Per scaricare la chiave, selezionare **Copia chiave negli Appunti.**
5. Nel riquadro destro selezionare **Copia** e quindi chiudere il riquadro.
6. Incollare la chiave in un file in un percorso sicuro e quindi immetterla come indicato durante l'installazione del software. La chiave è necessaria per attivare il software scaricato.
