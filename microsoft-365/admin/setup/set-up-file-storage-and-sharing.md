# <a name="set-up-file-storage-and-sharing-in-microsoft-365"></a>Configurare l'archiviazione e la condivisione di file in Microsoft 365

Uno dei modi migliori per configurare l'archiviazione e la condivisione di file per la tua azienda è usare OneDrive e un sito del team insieme. Questa soluzione è ideale per le piccole aziende con pochi dipendenti.

Guarda un breve video sull'archiviazione di file nel Microsoft 365 cloud.<br><br>

Se il video è stato utile, consultare la [serie di formazione completa per piccole imprese e nuovi utenti di Microsoft 365](../../business-video/index.yml).

## <a name="start-using-onedrive-and-your-team-site"></a>Iniziare a usare OneDrive e il sito del team

### <a name="team-members-can-store-their-own-files-in-onedrive"></a>I membri del team possono archiviare i propri file in OneDrive

Ogni persona della tua azienda a cui è assegnata Microsoft 365 licenza (e SharePoint Online selezionata) ottiene OneDrive cloud storage. Possono archiviare file relativi all'azienda qui per l'accesso da qualsiasi dispositivo e sono disponibili solo per quell'utente. Si può ad esempio archiviare una bozza di proposta, le note di una riunione o il testo di una demo da presentare.
  
I dipendenti possono anche condividere OneDrive file e cartelle. Se un dipendente è fuori sede o lascia la società, altri utenti possono accedere ai file condivisi archiviati in OneDrive (OneDrive condivisione delle cartelle del team).
  
Ecco come ogni persona del tuo team può configurare i file OneDrive e condividere file.

1. Passare al <a href="https://portal.office.com/ " target="_blank">portale Microsoft 365 e</a>accedere con il nome utente e la password.

2. Nell'icona di avvio delle app selezionare **OneDrive**.

3. In OneDrive, i membri del team possono archiviare i propri file aziendali. È possibile condividere singoli file o un'intera cartella. Selezionare un file o una cartella, fare clic con il pulsante destro del mouse e quindi **scegliere Condividi**.

    ![Condivisione di una cartella](../../media/e8df9df3-aea5-404d-a320-92d7826c260c.png)
  
4. Nella pagina **Invia collegamento** lasciare la selezione predefinita Chiunque abbia il collegamento può visualizzare **e modificare**.

    Digitare nomi o indirizzi di posta elettronica dei membri del team a cui si desidera accedere alla cartella e aggiungere un messaggio facoltativo.

    Se vuoi una copia personale dell'e-mail che verrà inviata, aggiungi il tuo indirizzo email all'elenco.

    ![Finestra di dialogo Condividi un collegamento che mostra la digitazione e la selezione di un nome.](../../media/877e6587-db9d-4903-a87b-11e570eee926.png)
  
5. Al termine dell'immissione di chi si desidera condividere, selezionare **Invia**. Il messaggio viene inviato immediatamente alle persone invitate.

    ![Condividere un collegamento con l'elenco dei nomi](../../media/e85625ea-7655-43f3-8623-72db68d0ea39.png)
  
6. Ecco l'aspetto dell'e-mail. 

    ![Posta elettronica con collegamento per condividere OneDrive cartella](../../media/750c92e1-f14f-404c-a6a3-2095e26c680c.png)
  
### <a name="upload-files-to-a-team-site-for-online-collaboration"></a>Upload file in un sito del team per la collaborazione online

I siti del team sono disponibili in un luogo in cui archiviare i file, denominato raccolta documenti.  
  
Ecco i passaggi per aggiungere file:
  
1. Nella home page del sito del team scegliere **Documenti** dal menu di spostamento a sinistra. Verrà visualizzata la **raccolta Documenti**.
  
2. Mentre hai ancora effettuato l'accesso Microsoft 365, apri Windows **Esplora file dalla** barra delle applicazioni o da un altro percorso. Passare ai file da caricare nel sito del team.

3. Selezionare i file da caricare nel sito del team e quindi trascinarli nella **raccolta documenti**.
  
4. Al termine, i file saranno archiviati sia nel sito del team che nel computer.
  
5. È possibile eliminare i file dal computer. Nel passaggio successivo, [Sincronizza file online con il PC o il Mac](#sync-online-files-with-your-pc-or-mac), si creerà una nuova posizione per questi file nel computer.

    Se si dispone di numerosi file o file di grandi dimensioni da caricare nel sito del team, leggere questi suggerimenti [sul caricamento di file di grandi dimensioni o di più file in una raccolta](https://support.microsoft.com/office/da549fb1-1fcb-4167-87d0-4693e93cb7a0#__toc384119242).

    Se serve più spazio di archiviazione, vedere [Modificare lo spazio di archiviazione per l'abbonamento](../../commerce/add-storage-space.md).

### <a name="sync-online-files-with-your-pc-or-mac"></a>Sincronizzare file online con il PC o il Mac

Ora che i file sono archiviati nel sito del team, è possibile impostarne la sincronizzazione con il PC o il Mac. In questo modo, puoi lavorare sui tuoi file dal TUO PC o Mac invece di lavorare in Microsoft Edge, Chrome o un altro browser. Inoltre, è utile avere una versione dei file sincronizzata con il computer per le situazioni in cui è necessario accedere a un file ma manca la connessione Internet.
  
Dopo aver impostato i file da sincronizzare con il computer, quando si è connessi a Internet, questi verranno sincronizzati automaticamente.
  
Ecco come sincronizzare i file archiviati nel sito del team con il desktop:
  
1. Nella home page del sito del team scegliere **Documenti** dal menu di spostamento a sinistra. In questo modo si viene alla raccolta Documenti.

    > [!TIP]
    > Quando si sincronizzano i file nel sito del team, si sincronizzano ogni raccolta file nel sito, non l'intero sito.
  
2. Scegliere **Sincronizza** per sincronizzare tutti i file. In questo caso, passare alla cartella specifica da sincronizzare.

3. Se viene richiesto di cambiare app, scegliere **Sì**. OneDrive è il processo che fa la sincronizzazione.
  
4. Se quindi si ottiene un **prompt di configurazione OneDrive,** accedere con l'account aziendale o dell'istituto di istruzione.

    ![OneDrive di configurazione](../../media/82cbb1ac-2ac5-42bd-82de-ba710bf46145.png)
  
5. Se non hai ancora sincronizzato il tuo OneDrive, potresti vedere una **schermata questa è la OneDrive della cartella.** Controllare il percorso in **La OneDrive cartella è qui**. Scegliere **Cambia percorso** se si desidera utilizzare un percorso diverso e quindi Selezionare **Avanti**.

    ![Modificare la cartella locale in questa schermata](../../media/6395485a-e729-4a9a-8e7d-b35e662435da.png)
  
6. I file nei siti del team verranno visualizzati nel riquadro sinistro di Esplora file sotto il nome dell'organizzazione. I file in OneDrive appariranno sotto "OneDrive - \<Name of Organization\> "

    ![Scopri cosa è stato sincronizzato nella cartella locale](../../media/93e2ca9f-4b5b-4930-a94d-ebc5b95aca84.png)
  
7. Testare la sincronizzazione aprendo un file nella cartella del team nel computer. Apportare una modifica e scegliere **Salva**.

## <a name="best-practices-for-file-storage-and-sharing"></a>Procedure consigliate per l'archiviazione e condivisione di file

Ecco alcuni suggerimenti per ottenere il massimo da OneDrive o dal tuo SharePoint del team.
  
### <a name="file-storage-and-collaboration-recommendations-for-other-types-of-small-businesses"></a>Suggerimenti sulla collaborazione e l'archiviazione di file per altri tipi di piccole aziende

- **Ditta individuale:** utilizza OneDrive per archiviare i tuoi file e condividerli con i clienti caso per caso.

- **Comproprietà:** entrambi i proprietari utilizzano OneDrive e condividono file avanti e indietro.

- **Aziende con clienti o partner esterni che hanno bisogno di accedere** ai file: creare un nuovo sito del team per archiviare e condividere documenti destinati a un cliente specifico. Configurare il sito per consentire l'accesso solo a tale cliente. Non devi quindi preoccuparti che un cliente avrà accidentalmente accesso alle informazioni destinate a un altro cliente.

### <a name="keep-private-files-private"></a>Mantenere privati i file riservati

Quando archivi un file in OneDrive accessibile solo da te, a meno che tu non lo conse ne conserti con altri. Quando condividi file, puoi scegliere di creare un collegamento che può essere inoltrato o di condividere solo con persone specifiche. È anche possibile creare cartelle distinte in OneDrive per scopi diversi, ad esempio cartelle pubbliche, private o per singoli progetti. Ogni cartella può essere condivisa con una persona o un gruppo diverso oppure con nessuno.
  
Per ulteriori informazioni sulla condivisione, vedere Anche [Condividere file e cartelle con Microsoft 365](https://support.microsoft.com/office/72f26d6c-bf9e-432c-8b96-e3c2437f5b65).
  
### <a name="track-how-much-space-you-have-left"></a>Tenere traccia dello spazio di archiviazione residuo

Per visualizzare la quantità di spazio di archiviazione rimasta in OneDrive, vedere [Gestire l'archiviazione OneDrive for Business archiviazione](https://support.microsoft.com/office/31519161-059C-4764-B6F8-F5CD29F7FE68).
  
### <a name="what-files-can-be-stored-in-onedrive-and-a-team-site"></a>Quali file possono essere archiviati in OneDrive e in un sito del team?

Sebbene sia possibile caricare quasi tutti i tipi di file, alcuni nomi di file e caratteri nei nomi di file non sono consentiti. Per ulteriori informazioni, vedere [Caratteri di file e tipi di file non validi in OneDrive for Business](https://support.microsoft.com/office/64883A5D-228E-48F5-B3D2-EB39E07630FA).
  
### <a name="enable-or-disable-third-party-storage-services"></a>Abilitare o disabilitare servizi di archiviazione di terze parti

È possibile abilitare l'archiviazione di terze parti per gli utenti in Microsoft 365 in modo che possano archiviare e condividere documenti utilizzando servizi come Dropbox oltre a siti OneDrive e del team. Può essere un'ottima soluzione per fornire servizi che gli utenti stanno forse già usando o preferiscono usare per i progetti commerciali. Se non si vuole che gli persone dell'organizzazione che usano Office aprano file in un servizio di terze parti, eseguire la procedura seguente per disattivarlo.
  
> [!IMPORTANT]
> L'archiviazione di terze parti è abilitata per impostazione predefinita, quindi è necessario eseguire immediatamente questi passaggi se non si desidera che sia disponibile per gli utenti.
  
1. Accedere all'interfaccia di <a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">amministrazione</a>.

2. Passare alla pagina **Impostazioni** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2053743" target="_blank">**impostazioni dell'organizzazione.**</a>

3. Nella scheda **Servizi** selezionare **Office sul web**.

4. Selezionare o deselezionare la casella di controllo per attivare o disattivare l'archiviazione di terze parti, quindi selezionare **Salva modifiche**.
