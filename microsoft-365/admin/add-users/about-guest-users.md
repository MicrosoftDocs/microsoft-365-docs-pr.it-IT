# <a name="guest-users-in-microsoft-365-admin-center"></a>Utenti guest nell'interfaccia di amministrazione di Microsoft 365

Tutti gli utenti che si aggiungono a Microsoft teams, SharePoint o Azure Active Directory vengono aggiunti anche all'elenco **Users Guest** nell'interfaccia di amministrazione di Microsoft. Gli utenti possono partecipare a riunioni, visualizzare documenti e chattare nei team a cui sono invitati.
Quando un utente viene visualizzato nell'elenco **utenti Guest** , è possibile rimuoverne l'accesso.

Per visualizzare gli utenti guest, nell'interfaccia di amministrazione di Microsoft 365, nel NAV sinistro, espandere **utenti** e quindi scegliere **utenti Guest**.

## <a name="add-guests-in-azure-active-directory"></a>Aggiungere gli utenti in Azure Active Directory

Per aggiungere gli utenti in Azure Active Directory, vedere [Add Guest Users](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal).

Dopo aver aggiunto un utente, è anche possibile assegnarlo a un gruppo o concedergli l'accesso a un'app nell'organizzazione. Dopo aver aggiunto un utente nel portale di Azure AD, l'utente verrà elencato nella pagina **utenti Guest** nell'interfaccia di amministrazione di Microsoft 365.
Dopo l'aggiunta di un utente all'elenco **degli utenti Guest** , è possibile [aggiungerli ai gruppi](../create-groups/manage-guest-access-in-groups.md#add-guests-to-a-microsoft-365-group-from-the-admin-center) nell'interfaccia di amministrazione di Microsoft 365.

Per invitare più clienti a collaborare con l'organizzazione, vedere [aggiungere gli utenti in blocco](https://docs.microsoft.com/azure/active-directory/b2b/tutorial-bulk-invite) .


## <a name="remove-a-guest"></a>Rimozione di un Guest

1. Nell'interfaccia di amministrazione di Microsoft 365 espandere **utenti** e quindi scegliere **utenti Guest**.
1. Nella pagina **utenti Guest** scegliere l'utente che si desidera rimuovere e quindi scegliere **Elimina un utente**. 

Per rimuovere gli utenti nel portale di Azure AD, vedere [rimuovere un utente Guest e risorse](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal#clean-up-resources).
