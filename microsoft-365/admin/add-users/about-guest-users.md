# <a name="guest-users-in-microsoft-365-admin-center"></a>Utenti guest nell'interfaccia di amministrazione di Microsoft 365

Quando si aggiungono ospiti a Microsoft teams o SharePoint, tali utenti vengono aggiunti all'elenco degli utenti guest nell'interfaccia di amministrazione. È inoltre possibile aggiungere gli utenti guest in Azure Active Directory in modo che possano collaborare con gli utenti dell'organizzazione. Gli utenti guest possono partecipare alle riunioni, visualizzare documenti e chattare nei team a cui sono invitati.
Dopo aver aggiunto un utente all'elenco degli utenti guest, è possibile rimuoverne l'accesso.

Per visualizzare gli utenti guest, nell'interfaccia di amministrazione di Microsoft 365, nel NAV sinistro, espandere **utenti**e quindi scegliere **utenti Guest**.

## <a name="add-guest-users-in-azure-active-directory"></a>Aggiungere gli utenti guest in Azure Active Directory

Per aggiungere gli utenti guest in Azure Active Directory, vedere [aggiungere gli utenti Guest](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal).

Quando si aggiunge un utente, è anche possibile assegnarlo a un gruppo o concedergli l'accesso a un'app nell'organizzazione. Dopo aver aggiunto un UER nel portale di Azure AD, l'utente verrà elencato nella pagina utenti guest nell'interfaccia di amministrazione di Microsoft 365.
Dopo l'aggiunta di un utente all'elenco degli utenti guest, è possibile [aggiungerli ai gruppi](../create-groups/manage-guest-access-in-groups.md#add-guests-to-a-microsoft-365-group-from-the-admin-center) nell'interfaccia di amministrazione di Microsoft 365.

Per invitare più clienti a collaborare con l'organizzazione, vedere [aggiungere gli utenti guest in massa](https://docs.microsoft.com/azure/active-directory/b2b/tutorial-bulk-invite) .


## <a name="remove-a-guest-user"></a>Rimuovere un utente Guest

Per rimuovere un utente Guest:

1. Nell'interfaccia di amministrazione di Microsoft 365 espandere **utenti** e quindi scegliere **utenti Guest**.
1. Nella pagina **utenti Guest** scegliere l'utente che si desidera rimuovere e quindi scegliere **Elimina un utente**. 

Per rimuovere gli utenti nel portale di Azure AD, vedere [rimuovere un utente Guest e risorse](https://docs.microsoft.com/azure/active-directory/b2b/b2b-quickstart-add-guest-users-portal#clean-up-resources).
