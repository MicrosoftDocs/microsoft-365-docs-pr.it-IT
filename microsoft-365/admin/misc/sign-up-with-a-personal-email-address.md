# <a name="signing-up-for-office-365-with-a-personal-email-address"></a>Iscrizione a Office 365 con un indirizzo di posta elettronica personale

## <a name="when-you-skip-adding-a-custom-domain"></a>Quando si ignora l'aggiunta di un dominio personalizzato

  
Quando si esegue l'iscrizione a Office 365 con l'indirizzo di posta elettronica personale e si ignora l'aggiunta di [un dominio personalizzato](../get-help-with-domains/what-is-a-domain.md) , è possibile limitare l'accesso a determinate funzionalità. 
  
Se si sceglie di aggiungere un dominio personalizzato adesso, è possibile accedere a tutte le funzionalità Premium seguenti:
  
|**Funzionalità**|**Office 365 con posta elettronica personale**|**Office 365 con dominio personalizzato**|
|:-----|:-----|:-----|
|**OneDrive <sup>1, 2</sup>**| [OneDrive personale](https://onedrive.live.com/about/en-us/plans/)| [OneDrive for Business](https://onedrive.live.com/about/en-us/business/) |
|**Applicazioni di Office: Word, Excel, PowerPoint, OneNote, Outlook, Access (solo PC)**| Sì | Sì 
|**Applicazioni aziendali <sup>3</sup> : connessioni Microsoft, fatturazione, inserzioni, prenotazioni, MileIQ e gestione clienti di Outlook**| No | Sì 
|**Accesso all'interfaccia di amministrazione di Microsoft 365**| Accesso limitato (fatturazione, supporto e configurazione del dominio) | Sì
|**Aggiungere utenti**| No | Sì
|**Servizi di Office 365: Exchange Online, Skype for business, Microsoft teams, SharePoint Online**| No | Sì
|**Strumenti per la sicurezza e la conformità**| No | Sì
