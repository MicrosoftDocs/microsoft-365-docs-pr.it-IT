# <a name="signing-up-for-office-365-with-a-personal-email-address"></a>Registrazione per l'Office 365 con un indirizzo di posta elettronica personale

## <a name="when-you-skip-adding-a-custom-domain"></a>Quando si ignora l'aggiunta di un dominio personalizzato


Quando ti i sign up for Office 365 with your personal email address and skip adding [a custom domain](../get-help-with-domains/what-is-a-domain.md) for now, you limit your access to certain features.

Se si sceglie di aggiungere un dominio personalizzato ora, si ottiene l'accesso a tutte le funzionalità premium seguenti:

|**Caratteristica**|**Office 365 posta elettronica personale**|**Office 365 con dominio personalizzato**|
|:-----|:-----|:-----|
|**OneDrive <sup>1, 2</sup>**| [Informazioni OneDrive](https://onedrive.live.com/about/plans/)| [OneDrive for Business](https://onedrive.live.com/about/en-us/business/) |
|**Office applicazioni: Word, Excel, PowerPoint, OneNote, Outlook, Access (solo PC),**| Sì | Sì
|**Applicazioni aziendali <sup>3</sup> : Microsoft Bookings e MileIQ**| No | Sì
|**Accesso a interfaccia di amministrazione di Microsoft 365**| Accesso limitato (fatturazione, supporto e configurazione del dominio) | Sì
|**Aggiungere utenti**| No | Sì
|**Office 365 servizi: Exchange Online, Skype for Business, Microsoft Teams, SharePoint Online**| No | Sì
|**Strumenti di sicurezza e conformità**| No | Sì
