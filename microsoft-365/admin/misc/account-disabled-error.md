# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Ottenere un account disabilitato in Outlook sul Web

Se viene visualizzato l'errore in cui l' **account è stato disabilitato** `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` quando si tenta di aprire Outlook sul Web (in precedenza noto come Outlook Web App), è possibile che l'amministratore abbia disabilitato l'accesso a Outlook sul Web.

L'amministratore può correggere questo errore attenendosi alla procedura descritta nell'argomento [abilitare o disabilitare Outlook Web App per una cassetta postale](https://docs.microsoft.com/exchange/recipients-in-exchange-online/manage-user-mailboxes/enable-or-disable-outlook-web-app).
