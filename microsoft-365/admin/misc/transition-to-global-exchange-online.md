# <a name="update-your-mx-records-to-transition-to-the-global-exchange-online-service"></a>Aggiornare i record MX alla transizione al servizio globale di Exchange Online

1. Accedere al [portale di amministrazione di Microsoft 365](https://admin.microsoft.com)e passare a **Settings** > **Domains** Settings

2. Lo stato verrà visualizzato a destra per ogni dominio. Se i domini dell'organizzazione puntano a Microsoft Cloud Germany Exchange Online, è necessario aggiornare il record MX.

3. Fare clic sul dominio, quindi fare clic su **errori DNS rilevati, fare clic qui per visualizzare**.

4. Questa pagina avrà istruzioni per illustrare come correggere il record MX. Se il registrar del dominio utilizza la [connessione di dominio](../setup/add-domain.md#registrars-with-domain-connect), è possibile fare clic su "Risolvi i record" in primo piano. In caso contrario, è possibile seguire il collegamento nella procedura guidata per **istruzioni dettagliate** per il registrar.
