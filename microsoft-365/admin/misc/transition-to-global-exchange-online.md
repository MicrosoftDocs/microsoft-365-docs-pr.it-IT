# <a name="update-your-mx-records-to-transition-to-the-global-exchange-online-service"></a>Aggiornare i record MX per la transizione al servizio Exchange Online globale

1. Accedi a [interfaccia di amministrazione di Microsoft 365](https://admin.microsoft.com)e vai **a** Impostazioni  >  **Domini**

2. Lo stato verrà visualizzato sul lato destro di ogni dominio. Se i domini dell'organizzazione puntano a Microsoft Cloud Germany Exchange Online, dovrai aggiornare il record MX.

3. Fai clic sul dominio, quindi fai **clic su Errori DNS rilevati, fai clic qui per visualizzare**.

4. In questa pagina sono disponibili istruzioni per illustrare come correggere il record MX. Se il registrar del dominio utilizza [Domain Connessione](../setup/add-domain.md#registrars-with-domain-connect), è possibile fare clic su "Correggi i miei record" in alto. In caso contrario, è possibile seguire il collegamento della procedura guidata **per istruzioni** dettagliate per il registrar.
