# <a name="update-your-mx-records-to-transition-to-the-global-exchange-online-service"></a>Aggiornare i record MX per la transizione al servizio Exchange Online globale

1. Accedere al portale di amministrazione di [Microsoft 365](https://admin.microsoft.com)e passare a **Domini**  >  **impostazioni**

2. Lo stato verrà visualizzato sul lato destro di ogni dominio. Se i domini dell'organizzazione puntano a Microsoft Cloud Germania Exchange Online, è necessario aggiornare il record MX.

3. Fare clic sul dominio, quindi fare **clic su Errori DNS rilevati, fare clic qui per visualizzare**.

4. Questa pagina avrà istruzioni per mostrare come correggere il record MX. Se il registrar del dominio usa [Domain Connect,](../setup/add-domain.md#registrars-with-domain-connect)è possibile fare clic su "Correggi i miei record" nella parte superiore. In caso contrario, è possibile seguire il collegamento della procedura guidata alle **istruzioni dettagliate** per il registrar.
