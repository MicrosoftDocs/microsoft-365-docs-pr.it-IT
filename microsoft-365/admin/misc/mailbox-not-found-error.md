# <a name="getting-a-mailbox-not-found-error-in-outlook-on-the-web"></a>Errore di recupero di una cassetta postale non trovata in Outlook sul Web?

Se si utilizza Outlook sul Web e non è possibile trovare una cassetta postale per  **l'errore, l'account**  utilizzato per connettersi a Outlook sul Web non dispone di una licenza di Exchange Online e pertanto nessuna cassetta postale è associata all'account. L'amministratore può assegnare una licenza al tuo account seguendo questi passaggi:

1. Aprire [l'interfaccia di amministrazione di Microsoft 365](https://portal.office.com/adminportal/home#/homepage) e passare a Utenti attivi nella sezione Utenti e selezionare l'utente che sta visualizzato l'errore.  
2. Nella pagina utente che si  apre, passare alla sezione  Licenze e app, selezionare il valore percorso appropriato e assegnare una licenza contenente Exchange Online (espandere la licenza per visualizzarne i dettagli). Al termine, fare clic su **Salva modifiche.**
