# <a name="quick-help-types-of-users"></a>Guida rapida: Tipi di utenti

Nella pagina **Utenti attivi** nell'interfaccia di amministrazione di Microsoft 365 è possibile filtrare in base ai tipi di utenti seguenti. 
  
|||
|:-----|:-----|
|**Frequenza d'uso** <br/> |**Definizione** <br/> |
|Utenti con licenza  <br/> |A questi utenti è stata assegnata una licenza di Microsoft 365, ad esempio Microsoft 365 Business Standard o Office 365 Enterprise E3, in modo che possano usare i servizi di Microsoft 365.  <br/> |
|Accesso consentito  <br/> |Questi utenti possono accedere a Microsoft 365 per creare documenti, controllare la posta elettronica e così via. La maggior parte degli utenti dovrebbe poter eseguire l'accesso in qualsiasi momento.  <br/> |
|Accesso bloccato  <br/> |Questi utenti non possono accedere a Microsoft 365. Un esempio è un utente che ha lasciato l'azienda e che ha bloccato l'accesso a Microsoft 365.  <br/> |
|Utenti senza licenza  <br/> |Questi utenti non hanno una licenza di Microsoft 365, come Microsoft 365 Business Standard o Office 365 Enterprise E3, quindi non possono usare le funzionalità di Microsoft 365. Alcuni esempi sono gli amministratori che devono gestire solo Microsoft 365, i dipendenti che hanno lasciato l'azienda o le cassette postali condivise e delle risorse non associate alle persone.  <br/> |
|Utenti con errori  <br/> |Agli account di questi utenti sono associati errori che devono essere risolti.  <br/> |
|Amministratori fatturazione  <br/> |Questi utenti possono effettuare acquisti, gestire le sottoscrizioni, gestire i ticket di supporto e monitorare l'integrità dei servizi per conto del delegante.  <br/> |
|Amministratori globali  <br/> |Questi utenti hanno accesso a tutte le funzionalità di amministrazione. La persona che si i signs up a Microsoft 365 diventa un amministratore globale, ma è possibile avere più di uno, se si vuole. Gli amministratori globali sono gli unici amministratori che possono assegnare ruoli di amministratore ad altri.  <br/> |
|Amministratori password  <br/> |Questi utenti possono reimpostare le password, gestire le richieste di servizio e monitorare l'integrità dei servizi per conto del delegante.  <br/> |
|Amministratori dei servizi  <br/> |Questi utenti possono gestire le richieste di servizio e monitorare l'integrità dei servizi per conto del delegante.  <br/> |
|Amministratori Gestione utenti  <br/> |Questi utenti possono reimpostare le password, monitorare l'integrità dei servizi, gestisce account utente, gruppi di utenti e richieste di servizi. Non possono eliminare un amministratore globale, creare altri ruoli di amministratore né reimpostare le password per gli amministratori fatturazione, gli amministratori globali e gli amministratori dei servizi.  <br/> |
|Aggiungi filtro personalizzato  <br/> |Con questa opzione, è possibile creare un filtro personalizzato per visualizzare solo determinati tipi di utenti di propria preferenza, ad esempio quali utenti sono nel piano Microsoft 365 Apps for business. [Guida rapida: Filtrare gli utenti.](https://docs.microsoft.com/microsoft-365/admin/add-users/create-edit-or-delete-a-custom-user-view)  <br/> |
