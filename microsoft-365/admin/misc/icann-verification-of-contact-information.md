# <a name="icann-verification-of-contact-information"></a>Verifica ICANN delle informazioni di contatto

Quando si acquista un dominio o si apportano modifiche alle informazioni di contatto per un dominio, il registrar deve convalidare le informazioni di contatto per conformarsi alle normative ICANN.

Per la convalida, viene inviato un messaggio di posta elettronica all'indirizzo del registrante.

 **Per verificare le informazioni di contatto, selezionare il collegamento nel messaggio di posta elettronica di verifica.** Se i proprietari del dominio non verificano l'indirizzo di posta elettronica, il dominio verrà sospeso. Passare quindi alla pagina [Domini nell'interfaccia](https://admin.microsoft.com/adminportal/home?ref=Domains) di amministrazione e controllare lo stato del dominio.
