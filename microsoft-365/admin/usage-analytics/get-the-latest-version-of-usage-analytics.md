# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Ottenere la versione più recente di analisi di utilizzo di Microsoft 365

L'app modello può essere aggiornata con nuovi dati o nuove visualizzazioni diverse volte all'anno. L'istanza esistente continuerà a funzionare, ma se si desidera ottenere la versione più recente, è necessario creare una nuova istanza e tutte le personalizzazioni devono essere applicate alla nuova istanza. Vedere [Enable Microsoft 365 Usage Analytics](enable-usage-analytics.md).
