# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Ottenere la versione più recente dell'Microsoft 365 di utilizzo

L'app modello può essere aggiornata con nuovi dati o nuove visualizzazioni più volte all'anno. L'istanza esistente continuerà a funzionare, ma se si desidera ottenere la versione più recente, è necessario creare una nuova istanza e applicare eventuali personalizzazioni alla nuova istanza. Vedere [Enable Microsoft 365 usage analytics](enable-usage-analytics.md).
