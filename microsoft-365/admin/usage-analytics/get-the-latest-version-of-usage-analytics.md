# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Ottenere la versione più recente dell'analisi dell'utilizzo di Microsoft 365

L'app modello può essere aggiornata con nuovi dati o nuove visualizzazioni diverse volte all'anno. L'istanza esistente continuerà a funzionare, ma se si desidera ottenere la versione più recente, è necessario creare una nuova istanza e applicare eventuali personalizzazioni alla nuova istanza. Vedere [Abilitare l'analisi dell'utilizzo di Microsoft 365.](enable-usage-analytics.md)
