# <a name="manage-industry-news"></a>Gestire le notizie del settore

Per fornire agli utenti titoli di notizie aggiornate sul settore e informazioni [](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews) dell'organizzazione, utilizzare il servizio Notizie per abilitare un feed di notizie personalizzato per l'organizzazione.Puoi anche gestire le impostazioni per la home page di Bing e la pagina Nuova scheda di Microsoft Edge (a partire dalla versione di Edge 87).

## <a name="what-your-users-will-see"></a>Cosa verranno visualizzati dagli utenti
 
Gli utenti connessi che visitano la home page di Bing visualizzano il news feed del settore sotto le informazioni personalizzate per l'organizzazione.   
 :::image type="content" source="../../media/manage-industry-news-2.jpg" alt-text="Screenshot of image carousel with industry news from the web":::
Possono anche visualizzare notizie aziendali, del settore e interne o informazioni di lavoro personalizzate nella pagina Nuova scheda di Microsoft Edge. 

:::image type="content" source="../../media/manage-industry-news-3.png" alt-text="Microsoft nella home page delle notizie":::

## <a name="news-settings"></a>Impostazioni delle notizie

Gli amministratori possono controllare le impostazioni dei news feed per l'organizzazione, tra cui il settore selezionato e la home page di Bing, insieme alla pagina Nuova scheda di Microsoft Edge (a partire dal rilascio di Edge 87).

1. Nell'interfaccia di amministrazione di Microsoft 365 passare a **Impostazioni**  >  **organizzazione**  >  **Impostazioni Servizi**  >  [Notizie](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews)
2. Nel riquadro **Notizie** fare clic sulla **scheda** Generale
3. **Nell'elenco Settore** selezionare i settori dell'organizzazione. Ciò determina le notizie generali visualizzate nel nuovo feed dell'organizzazione. Microsoft potrebbe pre-selezionare un settore usando le informazioni del tuo account. È possibile rimuovere o aggiungere settori aggiornando l'elenco Dei settori.
4. Nel campo **Argomenti** immettere gli argomenti di cui si desidera visualizzare gli articoli di news. Gli utenti non saranno in grado di modificare questi argomenti.
5. È possibile bloccare gli articoli contenenti parole chiave **nel campo Escludi** contenuto.  Ad esempio, per evitare che gli articoli contenenti la parola chiave "bake" non vengono visualizzati nel news feed, aggiungere la parola chiave "bake" nel campo Escludi contenuto. Evitare di includere termini generali (il, il contenuto e così via), possono impedire la visualizzazione di contenuto pertinente nei news feed aziendali.
6. Selezionare **Salva**. La visualizzazione delle modifiche potrebbe richiedere fino a 24 ore.

## <a name="bing-homepage"></a>Home page di Bing

Puoi personalizzare la home page di Bing per includere notizie sul tuo settore. 

1. Nell'interfaccia di amministrazione di Microsoft 365 passare a **Impostazioni**  >  **organizzazione**  >  **Servizi**  >  [Notizie.](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews) 
2. Nel riquadro **Notizie** fare clic sulla scheda **della home page di Bing** e selezionare Includi nella home page di **Bing.**
3. Le notizie del settore vengono visualizzate sotto le informazioni personalizzate dell'organizzazione in Bing.com.

## <a name="microsoft-edge-new-tab-page"></a>Pagina Nuova scheda di Microsoft Edge 
Quando gli utenti a questo tipo di accesso a Microsoft Edge (versione 87 o successiva) con un account aziendale o dell'istituto di istruzione valido, possono visualizzare notizie personalizzate per l'organizzazione.

1. Nell'interfaccia di amministrazione di Microsoft 365 passare a **Impostazioni**  >  **organizzazione**  >  **Servizi**  >  [Notizie.](https://admin.microsoft.com/adminportal/home?#/Settings/Services/:/Settings/L1/BingNews)
2. Nel riquadro **Notizie,** fare clic sulla **pagina Nuova scheda di Microsoft Edge.**
3. Selezionare **Consenti contenuto di Office 365 nella pagina Nuova scheda.** Se abilitati, gli utenti possono personalizzare la nuova scheda per visualizzare le informazioni di Office 365, inclusi i file consigliati e recenti, insieme ai siti di SharePoint usati di frequente e ad altre informazioni.
4. Selezionare **Mostra informazioni aziendali e notizie di settore nella pagina Nuova scheda.** Gli articoli relativi all'organizzazione e al settore verranno visualizzati per gli utenti che scelgono di visualizzare gli articoli nella pagina Nuova scheda.
