# <a name="search-in-the-microsoft-365-admin-center"></a>Eseguire ricerche nell'interfaccia di amministrazione di Microsoft 365

L'amministratore di un'Microsoft 365, è possibile utilizzare la ricerca per trovare utenti, eseguire azioni, passare a impostazioni diverse e leggere la documentazione. Con le nuove funzionalità di ricerca, la velocità di ricerca è migliorata ed è ora possibile eseguire ricerche da ogni pagina dell'interfaccia di amministrazione. La casella di ricerca è stata spostata nell'area banner nella parte superiore dell'interfaccia di amministrazione. È possibile utilizzare il **collegamento ALT+S** per utilizzare la ricerca da qualsiasi pagina.

:::image type="content" source="../../media/search-bar.png" alt-text="Screenshot: Search bar in the Microsoft 365 admin center":::

I risultati della ricerca sono organizzati in categorie diverse. La maggior parte delle categorie sono elementi nell'interfaccia di amministrazione. Ad esempio, utenti, gruppi, cassette postali condivise o domini. Altre categorie mostrano i luoghi a cui puoi passare, le azioni che puoi eseguire o le impostazioni a livello di app che puoi modificare. Esiste anche una categoria correlata alla documentazione.

Nelle sezioni seguenti vengono descritte le diverse aree e categorie dell'interfaccia di amministrazione in cui è possibile eseguire ricerche.

## <a name="before-you-begin"></a>Prima di iniziare

È necessario essere un amministratore per eseguire ricerche nell'interfaccia di amministrazione. L'ambito dei risultati della ricerca è quello delle autorizzazioni di amministratore dell'utente connesso. Ad esempio, se SharePoint'amministratore non visualizza un'area o una categoria nell'interfaccia di amministrazione, non la visualizza nella ricerca.

## <a name="users"></a>Utenti

Gli utenti possono essere trovati per nome visualizzato, cognome, nome, nome utente, indirizzo di posta elettronica principale o alias di posta elettronica. Selezionare la modifica del nome dell'utente per modificare i dettagli dell'utente.

:::image type="content" source="../../media/User-search-stock-picture-domain-blanked.png" alt-text="Screenshot: Search for a user Felix Henderson":::

Se si seleziona il menu con tre puntini (altre azioni) accanto al nome, è possibile reimpostarne la password.

:::image type="content" source="../../media/User-search-stock-picture-reset-password-domain-blanked.png" alt-text="Screenshot: User search with the option to reset password from search result":::

**Suggerimenti migliorare i risultati della ricerca degli utenti**

- Assicurati di scrivere correttamente i nomi degli utenti in quanto le ricerche degli utenti corrispondono esattamente alle proprietà menzionate in precedenza. Ad esempio, nell'esempio precedente Jus o Malz funzioneranno, ma un errore di ortografia, ad esempio, Jostin anziché Justin non troverà questo utente.
- Per ottenere una corrispondenza esatta, cercare in base all'indirizzo di posta elettronica principale o al nome utente.

## <a name="groups"></a>Gruppi

È possibile cercare Gruppi in base al nome del gruppo o all'indirizzo di posta elettronica del gruppo. È possibile selezionare il gruppo e modificare il gruppo da qualsiasi pagina.

:::image type="content" source="../../media/search-for-groups-mac.png" alt-text="Screenshot: Search results for groups in the admin center":::

**Suggerimenti migliorare i risultati della ricerca di gruppo**

Assicurati di digitare correttamente il nome del gruppo.

## <a name="actions"></a>Azioni

È possibile cercare la categoria Azioni che contiene le azioni usate di frequente nell'interfaccia di amministrazione di M365. Pensa alle azioni come verbo nel sistema. Ad esempio, è anche possibile cercare "reimpostare la password" da qualsiasi pagina e quindi reimpostare una o più password per gli utenti. È possibile cercare "eliminare un utente" ed eliminare l'utente dalla pagina Elimina utente.

:::image type="content" source="../../media/search-for-actions-mac.png" alt-text="Screenshot: Search results showing actions you can perform in the admin center":::

## <a name="navigation"></a>Navigazione

Risultati consente di passare rapidamente a una pagina specifica nell'interfaccia di amministrazione. Ad esempio, la ricerca di RBAC consente di accedere alla pagina Ruoli per i ruoli di Azure AD.

:::image type="content" source="../../media/search-for-navigation-mac.png" alt-text="Screenshot: Search results showing navigation path in the admin center":::

## <a name="settings"></a>Impostazioni

Cercare le impostazioni a livello di app supportate relative all'organizzazione, ai servizi sottoscritti e alle impostazioni di sicurezza e privacy.

:::image type="content" source="../../media/search-for-settings-mac.png" alt-text="Screenshot: Search results showing settings in the admin center":::

## <a name="domain"></a>Dominio

È possibile trovare collegamenti rapidi ai domini e quindi il collegamento consente di accedere alla pagina di panoramica del dominio.

:::image type="content" source="../../media/search-for-domains-mac.png" alt-text="Risultati della ricerca che mostrano i domini di proprietà dell'interfaccia di amministrazione":::

## <a name="documentation"></a>Documentazione

Una ricerca nella documentazione fornisce la documentazione della Guida pertinente in base alla frase di ricerca. Fai clic sull'argomento per altre informazioni.

:::image type="content" source="../../media/search-for-docs-mac.png" alt-text="Screenshot: Search showing documentation results in the admin center":::

## <a name="send-us-feedback"></a>Inviare feedback

Utilizzare questa sezione per inviare commenti e suggerimenti sull'esperienza di ricerca. Non possiamo rispondere a tutti i feedback, ma li leggiamo tutti e usiamo il tuo feedback per migliorare l'esperienza di ricerca. Assicurati di fornire il maggior numero di dettagli possibile nei commenti e suggerimenti.

:::image type="content" source="../../media/search-feedback-mac.png" alt-text="Screenshot: Send feedback on search in the admin center":::
