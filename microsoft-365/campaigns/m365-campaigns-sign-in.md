# <a name="sign-in-to-microsoft-365"></a>Accedere a Microsoft 365

## <a name="for-business-or-campaign-admins"></a>Per gli amministratori delle aziende o della campagna

Se si è connessi a Microsoft 365, si è l'amministratore di Microsoft 365. Ecco come eseguire l'accesso:

1. Trovare il nome utente e la password inviati all'indirizzo di posta elettronica fornito al passaggio 2 della procedura [di iscrizione.](m365-campaigns-sign-up.md#steps-to-sign-up)
2. Nel browser passare all'interfaccia di amministrazione di Microsoft 365 all'indirizzo <a href="https://go.microsoft.com/fwlink/p/?linkid=837890" target="_blank">https://admin.microsoft.com</a> .
3. Digita nome utente e password. Selezionare **Accedi**.
4. Nell'angolo in alto a destra della pagina, trova **l'anteprima nel** controllo. Seleziona **Anteprima in** modo da poter usare tutti i controlli descritti in Protezione avanzata per la [campagna.](m365-campaigns-security-overview.md)

## <a name="for-staff"></a>Per il personale

Configurare il personale come descritto in [Aggiungere utenti](../business/add-users-m365b.md?toc=/microsoft-365/campaigns/toc.json) È anche possibile reimpostare e inviare di nuovo le password nella **pagina Aggiungi** utenti.
Tutto il personale può accedere a <a href="https://office.com" target="_blank">https://Office.com</a> .
