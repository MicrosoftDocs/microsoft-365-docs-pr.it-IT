# <a name="share-files-and-videos"></a>Condividere file e video

Quando è necessario controllare chi può visualizzare e chi può modificare i file, è necessario archiviare i file in un percorso sicuro, in cui verificare che le autorizzazioni siano applicate in modo appropriato. È possibile utilizzare Microsoft Teams per archiviare i file e quindi condividerli all'interno o all'esterno dell'azienda, della pratica o della campagna tramite Microsoft Teams o inviando SharePoint collegamenti. L'invio di un collegamento anziché di un allegato di posta elettronica significa che si sa chi sta visualizzando e modificando i file e che non possono essere visualizzati o modificati senza l'autorizzazione dell'utente.

![Diagramma di una finestra Microsoft Teams, che mostra la scheda File e il collegamento Ottieni nel menu](../media/m365-democracy-teams-sharefiles.png)

Con i file in Microsoft Teams e SharePoint, è anche possibile lavorare sui file insieme e rivedere le modifiche reciprocamente. Usa Microsoft Teams per condividere file all'interno di una società, una pratica o una campagna. Se è necessario condividere esternamente con persone esterne all'organizzazione, è possibile aggiungerli come utenti guest a un team o inviare loro un collegamento SharePoint sicuro.

Puoi anche usare Microsoft Stream per archiviare e condividere video internamente. Questi video non sono visualizzabili al pubblico, quindi sono ideali per la messaggistica interna della campagna.

## <a name="best-practices"></a>Procedure consigliate

Usa questi metodi per condividere file e video in modo sicuro:

1. Archiviare i file Microsoft Teams o SharePoint e assicurarsi che siano presenti solo le persone che hanno bisogno di accedere a tali file.
2. Quando si desidera condividere, non allegare file a un messaggio di posta elettronica. Scegli invece **Ottieni collegamento da** Microsoft Teams o SharePoint e invia il collegamento tramite posta elettronica.
3. Per condividere un file esternamente, aggiungere l'utente come guest al team o usare SharePoint per ottenere un collegamento sicuro per condividere solo quel file.
4. Usa Microsoft Stream per ospitare i video che vuoi visualizzare nella campagna.
5. Usa Microsoft Teams o SharePoint per archiviare i file video su cui il team deve collaborare o condividere.
