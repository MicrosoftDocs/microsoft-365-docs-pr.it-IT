# <a name="set-up-meetings"></a>Configurare le riunioni

Le riunioni in Microsoft Teams includono audio, video e condivisione. Inoltre, poiché sono online, sarà sempre disponibile uno spazio per riunioni (senza bisogno di una sala o di un proiettore), anche se il personale è distribuito geograficamente o lavora in remoto. Le riunioni di Microsoft Teams sono un ottimo modo per riunirsi con il personale sia all'interno che all'esterno dell'organizzazione. Non è necessario essere membri dell'organizzazione o avere un account per partecipare a una riunione. È possibile pianificare ed eseguire riunioni online con Microsoft Teams. Durante una riunione, è possibile condividere lo schermo, condividere file, assegnare attività e altro ancora. Le campagne politiche possono includere personale, volontari e ospiti esterni all'organizzazione nella riunione. Piccole aziende o pratiche possono incontrarsi con il proprio personale o incontrarsi con clienti o partner tramite Microsoft Teams.

Scaricare un'infografica in [PDF](https://go.microsoft.com/fwlink/?linkid=2078712) o [PowerPoint](https://go.microsoft.com/fwlink/?linkid=2079515) per una rapida panoramica su come partecipare o ospitare una riunione online con Microsoft Teams.

## <a name="best-practices"></a>Procedure consigliate

Seguire queste procedure consigliate per le riunioni online:

- Pianificare le riunioni online direttamente in Microsoft Teams. È possibile scegliere un team e un canale e Teams inviterà automaticamente i partecipanti a tale team o canale.
- Serve una riunione esemptuosa? If you're in a one-to-one chat, choose **Meet now** to start a video or audio call with the person you're chatting with.
