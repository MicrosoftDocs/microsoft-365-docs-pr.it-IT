# <a name="set-up-meetings"></a>Configurare le riunioni

Le riunioni in Microsoft Teams includono audio, video e condivisione. Inoltre, poiché sono online, avrai sempre uno spazio riunioni (senza bisogno di una sala o di un proiettore!), anche se il personale è distribuito geograficamente o lavora in remoto. Microsoft Teams riunioni sono un ottimo modo per riunirsi con il personale sia all'interno che all'esterno dell'organizzazione. Non è necessario essere membri dell'organizzazione o disporre di un account per partecipare a una riunione. È possibile pianificare ed eseguire riunioni online utilizzando Microsoft Teams. Durante una riunione, è possibile condividere lo schermo, condividere file, assegnare attività e altro ancora. Le campagne politiche possono includere personale, volontari e ospiti esterni all'organizzazione nella riunione. Le piccole imprese o le pratiche possono incontrarsi con il proprio personale o incontrare clienti o partner in Microsoft Teams.

## <a name="best-practices"></a>Procedure consigliate

Seguire queste procedure consigliate per le riunioni online:

- Pianificare le riunioni online direttamente Microsoft Teams. Puoi scegliere un team e un canale e invitare Teams i partecipanti a tale team o canale automaticamente.
- Serve una riunione esemptuosa? Se si è in una chat uno-a-uno, scegliere Riunione per avviare una chiamata video o audio con la persona con cui si sta chattando.
