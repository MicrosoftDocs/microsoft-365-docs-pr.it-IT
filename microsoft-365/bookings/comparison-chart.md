# <a name="comparison-chart-bookings-web-app-vs-bookings-teams-app"></a>Grafico di confronto: app Web Bookings e app Bookings Teams

L'app Bookings in Teams consente agli utilità di pianificazione di gestire le attività principali e modificare alcune impostazioni. Tuttavia, il Web Bookings include funzionalità e impostazioni avanzate non ancora disponibili nell'app Teams.

Le funzionalità dell'app Teams vengono aggiunte continuamente e continueremo ad aggiornare questo elenco. Per ulteriori informazioni, vedere la sezione Tutte le funzionalità **dell'app Web Bookings** originale disponibile in Microsoft Teams? nelle domande frequenti. [](bookings-faq.yml)

| Funzionalità | App Web Bookings | App Bookings Teams |
|:---|:---|:---|
| Creare nuovi calendari di prenotazione | Sì | Sì |
| Aggiungere e rimuovere personale da un calendario di prenotazione | Sì | Sì |
| Creare nuovi tipi di appuntamenti | Sì | Sì |
| Pianificare prenotazioni online | Sì | Sì |
| Modificare i dettagli aziendali | Sì | Sì |
| Aggiungere personale con autorizzazioni Guest per persone esterne all'organizzazione | Sì | No |
| Pianificare prenotazioni di persona o offline | Sì | No |
| Pianificare prenotazioni di gruppi o più clienti | Sì | No |
| Impostare un logo per l'azienda | Sì | No |
| Impostare l'orario di ufficio | Sì | No |
| Pubblicare una pagina di pianificazione in modalità self-service | Sì | No |
| Gestire i contatti dei clienti | Sì | No |
| Impostare il tempo di insodd per il personale | Sì | No |
