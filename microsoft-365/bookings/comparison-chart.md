# <a name="comparison-chart-bookings-web-app-vs-bookings-teams-app"></a>Grafico di confronto: Bookings Web App e App Bookings Teams

L'app Prenotazioni in Teams consente agli utilità di pianificazione di gestire le attività principali e modificare alcune impostazioni. Tuttavia, il Web bookings include funzionalità e impostazioni avanzate non ancora disponibili nell'app Teams.

Le funzionalità delle app di Teams vengono aggiunte continuamente e continueremo ad aggiornare questo elenco. Vedere la sezione Domande frequenti **sulle funzionalità dell'app Web** Prenotazioni [](bookings-faq.md) originale disponibile in Microsoft Teams.

| Funzionalità | App Web Bookings | App Bookings Teams |
|:---|:---|:---|
| Creare nuovi calendari di prenotazione | Sì | Sì |
| Aggiungere e rimuovere personale da un calendario delle prenotazioni | Sì | Sì |
| Creare nuovi tipi di appuntamenti | Sì | Sì |
| Pianificare prenotazioni online | Sì | Sì |
| Modificare i dettagli aziendali | Sì | Sì |
| Aggiungere personale con autorizzazioni Guest per persone esterne all'organizzazione | Sì | No |
| Pianificare prenotazioni di persona o offline | Sì | No |
| Pianificare prenotazioni di gruppi o multi-cliente | Sì | No |
| Impostare un logo per l'azienda | Sì | No |
| Impostare l'orario di ufficio | Sì | No |
| Pubblicare una pagina di pianificazione self-service | Sì | No |
| Gestire i contatti dei clienti | Sì | No |
| Impostare il tempo libero per il personale | Sì | No |
