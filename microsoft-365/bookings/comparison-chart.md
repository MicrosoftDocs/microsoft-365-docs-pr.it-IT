# <a name="comparison-chart-bookings-web-app-vs-bookings-teams-app"></a>Grafico di confronto: bookings Web App vs bookings teams app

L'app bookings in teams consente agli utilità di pianificazione di gestire le attività principali e modificare alcune impostazioni. Tuttavia, il sito Web bookings include funzionalità e impostazioni avanzate che non sono ancora disponibili nell'app teams.

Le funzionalità delle app per i team vengono aggiunte continuamente e continueremo a aggiornare questo elenco. [Per ulteriori](bookings-faq.md) informazioni, vedere la sezione **tutte le funzionalità dell'app Web bookings originale disponibile in Microsoft teams?** .

| Funzionalità | Bookings Web App | App squadre di prenotazioni |
|:---|:---|:---|
| Creare nuovi calendari di prenotazione | Sì | Sì |
| Aggiungere e rimuovere personale da un calendario di prenotazione | Sì | Sì |
| Creare nuovi tipi di appuntamenti | Sì | Sì |
| Pianificare le prenotazioni online | Sì | Sì |
| Modificare i dettagli aziendali | Sì | Sì |
| Aggiungere personale con autorizzazioni Guest per utenti esterni all'organizzazione | Sì | No |
| Pianificare prenotazioni in-person o offline | Sì | No |
| Pianificazione di gruppi o prenotazioni multi-cliente | Sì | No |
| Impostare un logo per la propria azienda | Sì | No |
| Impostare l'orario di ufficio | Sì | No |
| Pubblicare una pagina di pianificazione in modalità self-service | Sì | No |
| Gestire i contatti dei clienti | Sì | No |
| Impostare l'intervallo di tempo per il personale | Sì | No |
