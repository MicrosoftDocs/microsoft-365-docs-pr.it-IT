# <a name="reporting-info-for-bookings"></a>Informazioni sulla creazione di report per Bookings

Ora è possibile visualizzare una visualizzazione di 120 giorni del calendario di Booking in un file TSV. I file TSV sono file con valori separati da tabulazioni. È possibile visualizzare o modificare un file di questo tipo con qualsiasi editor di testo o foglio di calcolo, ad esempio Excel.

## <a name="see-120-days-booking-activity"></a>Vedere attività di prenotazione di 120 giorni

1. Nel dashboard del calendario di Booking, selezionare Esporta altri dati come TSV.

:::image type="content" source="../media/bookings-activity.png" alt-text="Screenshot: 30 days Booking activities":::

1. Salvare il file con un nuovo nome e specificare il formato xls o xlsx.

1. Apri il file per visualizzare la visualizzazione di 120 giorni del calendario di Booking.

Il file contiene una riga per ogni evento nel calendario delle prenotazioni per i 60 giorni precedenti e i prossimi 60 giorni. Ogni riga include le informazioni su data e ora per l'evento, il nome del cliente, l'e-mail del cliente, il numero di telefono del cliente, l'indirizzo del cliente, il personale assegnato, il servizio selezionato, la posizione e la durata dell'appuntamento.
