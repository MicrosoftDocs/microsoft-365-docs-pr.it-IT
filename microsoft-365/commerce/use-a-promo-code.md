## <a name="promo-code-terms"></a><span data-ttu-id="c272b-115">Condizioni per i codici promozionali</span><span class="sxs-lookup"><span data-stu-id="c272b-115">Promo code terms</span></span>

<span data-ttu-id="c272b-116">Le linee guida seguenti si applicano ai codici promozionali per gli abbonamenti a Microsoft 365 for business.</span><span class="sxs-lookup"><span data-stu-id="c272b-116">The following guidelines apply to promo codes for Microsoft 365 for business subscriptions.</span></span>
  
- <span data-ttu-id="c272b-117">**Numero di codici promozionali per abbonamento** È possibile usare un codice promozionale per più abbonamenti nello stesso ordine, ma non si possono usare più codici promozionali per lo stesso abbonamento.</span><span class="sxs-lookup"><span data-stu-id="c272b-117">**Number of promo codes per subscription** You can use a promo code for multiple subscriptions in the same order, but you cannot use multiple promo codes for the same subscription.</span></span> 
    
- <span data-ttu-id="c272b-p104">**Durata dello sconto** Dopo l'applicazione all'ordine, lo sconto associato al codice promozionale avrà una durata pari a quella dell'abbonamento. Se l'abbonamento è rinnovabile mensilmente, il codice di sconto sarà valido fino alla fine del mese di fatturazione in corso. If your subscription's term is yearly (even if it's paid monthly), the promo code discount lasts until the end of the billing year you entered it in.</span></span> 
    
- <span data-ttu-id="c272b-p105">**Limiti di licenza per i codici promozionali** Per alcuni codici promozionali è previsto un livello massimo. - <span data-ttu-id="c272b-p106">**Date di fine validità del codice promozionale** La data di fine della promozione è l'ultimo giorno entro il quale è possibile usare un codice promozionale per una specifica campagna.
