# <a name="apply-for-tax-exempt-status-for-office-365-for-business"></a>Richiedere l'esenzione fiscale per Office 365 per le aziende

Se si ha diritto all'esenzione fiscale nel proprio paese o nella propria area geografica, [contattare il supporto tecnico](../../admin/contact-support-for-business-products.md) per creare una richiesta di servizio per stabilire lo stato di esenzione fiscale per l'abbonamento. Quando si invia una richiesta, è necessario fornire la documentazione seguente: 
  
|**Posizione**|**Documentazione**|
|:-----|:-----|
|Stati Uniti e Canada  <br/> |Certificato IVA per uso commerciale  <br/> |
|Portorico  <br/> |Certificado de compras Exentas  <br/> |
|Irlanda  <br/> |Ter/56A-certificato di esenzione fiscale  <br/> |
|Organizzazioni internazionali che detengono esenzione fiscale  <br/> |Conferma della certificazione/lettera dalle autorità tributarie locali rilevanti  <br/> |
