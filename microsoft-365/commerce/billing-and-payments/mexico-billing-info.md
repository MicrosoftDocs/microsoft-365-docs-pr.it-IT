# <a name="billing-information-for-microsoft-365-for-business-in-mexico"></a>Informazioni di fatturazione per Microsoft 365 per le aziende in Messico

## <a name="1-where-can-i-get-an-invoice-for-my-microsoft-365-for-business-purchase"></a>1. Dove posso ottenere una fattura per l'acquisto di Microsoft 365 per le aziende?

La fattura viene inviata all'indirizzo di posta elettronica di notifica della fatturazione registrato 24 ore dopo la conferma dell'acquisto. Per scaricare la fattura nell'interfaccia di amministrazione di Microsoft 365, passare alla pagina **Fatture**&  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">pagamenti.</a>

> [!IMPORTANT]
> Se vuoi che la fattura includa la tua RFC, devi aggiungerla prima della conferma dell'acquisto.

## <a name="2-how-can-i-add-my-rfc-registro-federal-de-contribuyentes-to-the-invoice-i-get-for-the-purchase-of-microsoft-365-for-business"></a>2. Come posso aggiungere la mia RFC (Registro Federal de Contribuyentes) alla fattura che ottengo per l'acquisto di Microsoft 365 per le aziende?

### <a name="during-the-purchase-process-checkout"></a>Durante il processo di acquisto (checkout)

Durante il processo di acquisto della transazione, al passaggio **4. Pagamento e fatturazione**, selezionare la casella per immettere una RFC. Questo passaggio è necessario in modo da poterlo visualizzare nella fattura di acquisto.

### <a name="for-your-future-purchases"></a>Per gli acquisti futuri

È possibile aggiungere o modificare rfc in modo che le informazioni riflettano nei futuri acquisti di Microsoft 365 per le aziende. Per aggiungere o modificare rfc, passare alla pagina **Account**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank">fatturazione.</a>

> [!NOTE]
> Se si è in una sottoscrizione di fatturazione ricorrente mensile o annuale, l'aggiunta o la modifica di RFC si riflette sulla fattura dell'addebito successivo.

## <a name="3-can-i-add-or-modify-my-rfc-to-an-invoice-that-was-already-generated"></a>3. È possibile aggiungere o modificare la rfc a una fattura già generata?

È importante aggiungere o modificare rfc prima di effettuare l'acquisto. Dopo aver generato la fattura di Microsoft 365 per le aziende, non è possibile aggiungere o modificare le informazioni RFC.

## <a name="4-why-dont-i-get-a-cfdi-comprobante-fiscal-digital-por-internet"></a>4. Perché non si ottiene un CFDI (Comprobante Fiscal Digital por Internet)?

Microsoft Corporation, in quanto residente fiscale negli Stati Uniti d'America, è in grado di emettere fatture ai sensi delle leggi di giurisdizione e non è obbligata a fornire una fattura elettronica CFDI.

## <a name="5-is-the-invoice-i-receive-valid-for-my-accounting-operations-in-mexico"></a>5. La fattura ricevuta è valida per le operazioni contabili in Messico?

Una fattura generata da entità esterne (in questo caso Microsoft Corporation) che include la rispettiva RFC è valida a fini fiscali in Messico. Per ulteriori informazioni, è consigliabile consultare il commercialista.
