# <a name="billing-information-for-microsoft-365-for-business-in-mexico"></a>Informazioni di fatturazione per Microsoft 365 per le aziende in Messico

## <a name="1-where-can-i-get-an-invoice-for-my-microsoft-365-for-business-purchase"></a>1. Dove posso ottenere una fattura per il mio Microsoft 365 per l'acquisto aziendale?

La fattura viene inviata all'indirizzo e-mail di notifica di fatturazione registrato 24 ore dopo la conferma dell'acquisto. Per scaricare la fattura nell'Microsoft 365 di amministrazione, passare alla pagina **Fatturazione**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">& pagamenti.</a>

> [!IMPORTANT]
> Se vuoi che la fattura includa la tua RFC, devi aggiungerla prima della conferma dell'acquisto.

## <a name="2-how-can-i-add-my-rfc-registro-federal-de-contribuyentes-to-the-invoice-i-get-for-the-purchase-of-microsoft-365-for-business"></a>2. Come posso aggiungere il mio RFC (Registro Federal de Contribuyentes) alla fattura che ottengo per l'acquisto di Microsoft 365 per le aziende?

### <a name="during-the-purchase-process-checkout"></a>Durante il processo di acquisto (Checkout)

Durante il processo di acquisto dell'estrazione, quando si arriva al passaggio **4. Pagamento e fatturazione**, selezionare la casella per immettere una RFC. Questo passaggio è necessario in modo da poterlo visualizzare nella fattura di acquisto.

### <a name="for-your-future-purchases"></a>Per gli acquisti futuri

Puoi aggiungere o modificare la rfc in modo che le informazioni si riflettano in futuro Microsoft 365 per gli acquisti aziendali. Per aggiungere o modificare la RFC, passare alla **pagina Account di**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank">fatturazione.</a>

> [!NOTE]
> Se si è in una sottoscrizione di fatturazione ricorrente mensile o annuale, l'aggiunta o la modifica della RFC si riflette sulla fattura dell'addebito successivo.

## <a name="3-can-i-add-or-modify-my-rfc-to-an-invoice-that-was-already-generated"></a>3. È possibile aggiungere o modificare la rfc a una fattura già generata?

È importante aggiungere o modificare la RFC prima di effettuare l'acquisto. Dopo aver generato la fattura Microsoft 365 per le aziende, non è possibile aggiungere o modificare le informazioni RFC.

## <a name="4-why-dont-i-get-a-cfdi-comprobante-fiscal-digital-por-internet"></a>4. Perché non si ottiene un CFDI (Comprobante Fiscal Digital por Internet)?

Microsoft Corporation, in quanto residente fiscale negli Stati Uniti d'America, è in grado di emettere solo fatture ai sensi delle leggi di giurisdizione e non è obbligata a fornire una fattura elettronica CFDI.

## <a name="5-is-the-invoice-i-receive-valid-for-my-accounting-operations-in-mexico"></a>5. La fattura ricevuta è valida per le operazioni contabili in Messico?

Una fattura generata da entità estere (in questo caso Microsoft Corporation) che include il rispettivo RFC è valida a fini fiscali in Messico. Per ulteriori informazioni, ti consigliamo di consultare il tuo commercialista.
