# <a name="billing-information-for-microsoft-365-for-business-in-mexico"></a>Informazioni di fatturazione per Microsoft 365 for business in Messico

## <a name="1-where-can-i-get-an-invoice-for-my-microsoft-365-for-business-purchase"></a>1. dove è possibile ottenere una fattura per l'acquisto di Microsoft 365 per le aziende?

La fattura viene inviata all'indirizzo di posta elettronica di notifica di fatturazione registrata 24 ore dopo la conferma dell'acquisto. Per scaricare la fattura nell'interfaccia di amministrazione di Microsoft 365, **passare alla**  >  pagina <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">fatture & pagamenti</a> .

> [!IMPORTANT]
> Se si desidera che la fattura includa l'RFC, è necessario aggiungerlo prima della conferma dell'acquisto.

## <a name="2-how-can-i-add-my-rfc-registro-federal-de-contribuyentes-to-the-invoice-i-get-for-the-purchase-of-microsoft-365-for-business"></a>2. come è possibile aggiungere la RFC (Registro Federal de Contribuyentes) alla fattura ricevuta per l'acquisto di Microsoft 365 for business?

### <a name="during-the-purchase-process-checkout"></a>Durante il processo di acquisto (checkout)

Durante il processo di acquisto del checkout, quando si arriva al passaggio **4. Pagamento e fatturazione**, selezionare la casella per immettere un RFC. Questo passaggio è necessario affinché sia possibile visualizzarlo riflesso nella fattura di acquisto.

### <a name="for-your-future-purchases"></a>Per gli acquisti futuri

È possibile aggiungere o modificare l'RFC in modo che le informazioni siano riflesse nei futuri acquisti di Microsoft 365 per le aziende. Per aggiungere o modificare l'RFC, passare alla pagina **Billing**  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank">account Fatturazione</a> fatturazione.

> [!NOTE]
> Se si è in un abbonamento a fatturazione ricorrente mensile o annuale, l'aggiunta o la modifica dell'RFC viene applicata alla fattura dell'addebito successivo.

## <a name="3-can-i-add-or-modify-my-rfc-to-an-invoice-that-was-already-generated"></a>3. è possibile aggiungere o modificare l'RFC in una fattura già generata?

È importante aggiungere o modificare l'RFC prima di effettuare l'acquisto. Dopo aver generato la fattura di Microsoft 365 per le aziende, non è possibile aggiungere o modificare le informazioni RFC.

## <a name="4-why-dont-i-get-a-cfdi-comprobante-fiscal-digital-por-internet"></a>4. perché non si riceve un CFDI (comprobante Fiscal Digital por Internet)?

Microsoft Corporation, in quanto residente fiscale negli Stati Uniti d'America, è in grado di emettere fatture secondo le leggi di giurisdizione e non è obbligato a fornire una fattura elettronica CFDI.

## <a name="5-is-the-invoice-i-receive-valid-for-my-accounting-operations-in-mexico"></a>5. la fattura ricevuta è valida per le operazioni contabili in Messico?

Una fattura generata da entità esterne (in questo caso Microsoft Corporation) che include il rispettivo RFC è valida a fini fiscali in Messico. (Si consiglia di consultare il contabile per ulteriori informazioni).
