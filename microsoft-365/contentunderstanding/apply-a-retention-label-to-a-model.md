# <a name="apply-a-retention-label-to-a-document-understanding-model-preview"></a>Applicazione di un'etichetta di conservazione a un modello di comprensione del documento (anteprima)

> [!Note] 
> Il contenuto di questo articolo è relativo all'anteprima privata di Project Cortex. [Per ulteriori informazioni, vedere Project Cortex](https://aka.ms/projectcortex).

</br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4CSoL]

</br>

È possibile applicare facilmente un' [etichetta di conservazione](https://docs.microsoft.com/microsoft-365/compliance/retention?view=o365-worldwide#retention-labels) a un modello di comprensione del documento in Project Cortex.

Le etichette di conservazione consentono di applicare le impostazioni di conservazione ai documenti identificati dal modello.  Ad esempio, è possibile che il modello non identifichi solo i documenti di rinnovo del contratto caricati nella raccolta documenti, ma anche che gli applichi un tag di conservazione, in modo che tali documenti non possano mai essere eliminati.

È possibile applicare un'etichetta di conservazione al modello di comprensione dei documenti tramite le impostazioni del modello. 

> [!Note]
> Affinché le etichette di conservazione siano disponibili per il modello, è necessario [crearle e pubblicarle nel centro conformità di Microsoft 365](https://docs.microsoft.com/microsoft-365/compliance/create-apply-retention-labels?view=o365-worldwide#how-to-create-and-publish-retention-labels).
