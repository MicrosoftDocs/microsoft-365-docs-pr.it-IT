# <a name="introduction-to-sharepoint-syntex"></a>Introduzione a SharePoint Syntex


</br>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4CSu7] 

</br>

SharePoint Syntex è un servizio di Office 365 che consente alle organizzazioni di:

- Utilizzo dell'intelligenza artificiale avanzata e dell'insegnamento automatico per amplificare le esperienze umane
- Automatizzare l'elaborazione del contenuto
- Trasformare il contenuto in una conoscenza

Le risorse presenti in questa pagina sono state progettate per iniziare a conoscere e utilizzare SharePoint Syntex nell'organizzazione.
