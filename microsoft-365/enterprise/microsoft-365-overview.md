# <a name="microsoft-365-enterprise-overview"></a>Panoramica di Microsoft 365 Enterprise

Microsoft 365 Enterprise è una soluzione completa e intelligente che consente agli tutti si trovino creative e collaborare in modo protetto. 

Benché progettato per organizzazioni di grandi dimensioni, Microsoft 365 Enterprise è possibile utilizzare anche per le aziende di piccole e medie dimensioni che è necessario utilizzare le funzionalità più avanzate di protezione e produttività. 

## <a name="components"></a>Componenti

Microsoft 365 Enterprise è costituita da:

|||
|:-------|:-----|
| Office 365 Enterprise | Include entrambe Office 365 ProPlus, le applicazioni di Office più recenti per PC e Mac (ad esempio, Word, Excel, PowerPoint, Outlook e altri utenti) e una suite completa di servizi online per posta elettronica, la memorizzazione dei file e la collaborazione, riunioni e altre risorse. |
| Windows 10 Enterprise | Soddisfi le esigenze di organizzazioni di grandi e medie dimensioni, fornendo agli utenti il più efficiente e protetta versione di Windows e ai professionisti IT completi di gestione app, dispositivi e la distribuzione. |
| Sicurezza (EMS) + mobilità aziendale | Include Intune Microsoft, che è un servizio di gestione (EMM) mobilità aziendale basata su cloud che consente di abilitare forza lavoro per la produttività mantenendo la protezione dei dati aziendali. |
|||

## <a name="plans"></a>Piani

Microsoft 365 Enterprise è disponibile nei tre piani.

|||
|:-------|:-----|
| E3 | Include Enterprise di Office 365, Windows 10 Enterprise e mobilità aziendale + sicurezza (EMS). |
| E5 | Include tutte funzionalità dell'E3 plus sicurezza avanzata, vocali e gli strumenti di analisi dei dati. |
| F1 | Progettato appositamente per la connessione lavoratori firstline a questi strumenti e risorse necessarie per il loro lavoro migliore. Firstline lavoratori sono i primi nella riga a PC con i clienti e rappresentano marchio dell'azienda e il valore, |
|||

Per ulteriori informazioni, vedere [caratteristiche e funzionalità per ciascun piano](https://www.microsoft.com/microsoft-365/compare-all-microsoft-365-plans).

## <a name="deploying"></a>Distribuzione

Esistono due modi per distribuire i prodotti, funzionalità e componenti di Microsoft 365 Enterprise:

1. In collaborazione con FastTrack
  
   Con FastTrack, tecnici Microsoft semplificare lo spostamento nel cloud le proprie esigenze. Vedere [FastTrack per Microsoft 365](https://fasttrack.microsoft.com/microsoft365).
  
2. Eseguire tale operazione manualmente

   [Guida alla distribuzione di Microsoft 365 Enterprise](deploy-microsoft-365-enterprise.md) consente di passare step di step tramite creazione fuori dei carichi di lavoro per la produttività e l'infrastruttura. 

Per ulteriori informazioni sulla distribuzione, vedere Procedura:

- [Utenti di](deploy-microsoft-365-enterprise.md#how-customers-use-microsoft-365-enterprise) utilizzare Microsoft 365 Enterprise.
- [Microsoft](deploy-microsoft-365-enterprise.md#how-microsoft-uses-microsoft-365-enterprise) utilizza Microsoft 365 Enterprise.
- [Contoso Corporation](contoso-overview.md) ha distribuito Microsoft 365 Enterprise.
