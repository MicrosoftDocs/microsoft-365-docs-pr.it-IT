# <a name="plan-for-microsoft-365-enterprise"></a>Pianificare Microsoft 365 Enterprise

Quando si sposta un'organizzazione aziendale in Microsoft 365, è importante pianificare in anticipo e prendere le decisioni di progettazione chiave che semplificano la distribuzione IT e l'adozione da parte degli utenti. 

## <a name="planning-with-microsoft-365-fasttrack"></a>Pianificazione con Microsoft 365 FastTrack

[FastTrack per Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365) è il metodo migliore per ottenere assistenza da Microsoft per la pianificazione della distribuzione di Microsoft 365. FastTrack può essere utile per le considerazioni di progettazione più comuni e può rispondere alle domande lungo il percorso. 

>[!Note]
>È anche possibile ottenere assistenza da un [partner Microsoft.](https://www.microsoft.com/solution-providers/home)
>

## <a name="do-it-yourself-planning-for-microsoft-365"></a>Pianificazione fai-da-te per Microsoft 365

Per pianificare Microsoft 365 in modo personalizzato, eseguire le decisioni di pianificazione e progettazione per queste aree:

- Tenant di Microsoft 365

  Include la pianificazione delle connessioni di rete a Internet, delle identità di Microsoft 365 e dell'integrazione con app, locale, Azure e altri elementi. Per altre informazioni, vai a [Sottoscrizioni, licenze, account](subscriptions-licenses-accounts-and-tenants-for-microsoft-cloud-offerings.md) e tenant per le offerte cloud di Microsoft.

- Supporto per i client

  Include l'autenticazione basata su certificati, la gestione dei dispositivi mobili, le opzioni di autenticazione e la collaborazione tra tenant. Per ulteriori [informazioni, passare](microsoft-365-client-support-certificate-based-authentication.md) a Autenticazione basata su certificato.

- Supporto per l'autenticazione moderna ibrida

  Include la pianificazione dell'autenticazione moderna quando si utilizzano configurazioni ibride delle principali app di produttività di Microsoft 365. Per ulteriori [informazioni, vedere Panoramica dell'autenticazione](hybrid-modern-auth-overview.md) moderna ibrida e prerequisiti.

- Client e server office meno recenti

  Include informazioni sulla migrazione per i prodotti client e server di Office 2007 e Office 2010. Per ulteriori [informazioni, vedere Pianificare](plan-upgrade-previous-versions-office.md) l'aggiornamento.

È anche possibile accedere all'abbonamento a Microsoft 365 e usare le guide alla configurazione per i servizi di [Microsoft 365 e Office 365.](setup-guides-for-microsoft-365.md)
