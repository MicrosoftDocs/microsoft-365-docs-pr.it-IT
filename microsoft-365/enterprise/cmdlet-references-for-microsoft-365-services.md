# <a name="cmdlet-references-for-microsoft-365-services"></a>Riferimenti cmdlet per i servizi di Microsoft 365 

*Questo articolo si applica sia a Microsoft 365 Enterprise che a Office 365 Enterprise*.

In questo articolo vengono forniti riferimenti ai cmdlet per i vari servizi di Microsoft 365 e istruzioni di connessione per ogni servizio di Microsoft 365 che PowerShell supporta.
  
> [!NOTE]
> Per connettersi a tutti i servizi contemporaneamente, vedere [Connect to all Microsoft 365 services in a single Windows PowerShell window](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md).
  
## <a name="azure-active-directory-powershell-cmdlets"></a>Cmdlet di Azure Active Directory PowerShell

Gli argomenti di riferimento sui cmdlet di Azure Active Directory PowerShell per Graph sono disponibili nella sezione di riferimento della documentazione di [Azure Active Directory PowerShell per Graph.](https://docs.microsoft.com/powershell/azure/active-directory/install-adv2?view=azureadps-2.0)

Gli argomenti di riferimento sul modulo di Azure Active Directory Windows PowerShell cmdlet sono disponibili nella sezione Reference della documentazione di [Azure Active Directory (MSOnline).](https://docs.microsoft.com/powershell/azure/active-directory/overview?view=azureadps-1.0)

Per istruzioni sulla connessione di Microsoft 365 PowerShell, vedere Connettersi a [Microsoft 365 con PowerShell.](connect-to-microsoft-365-powershell.md)
  
## <a name="exchange-online-powershell-cmdlets"></a>Cmdlet di Exchange Online PowerShell

Gli argomenti di riferimento sui cmdlet di Exchange Online sono disponibili nella sezione di riferimento della documentazione [di PowerShell di Exchange Online.](https://docs.microsoft.com/powershell/exchange/exchange-online-powershell)
  
Per istruzioni sulla connessione per PowerShell di Exchange Online, vedere [Connettersi a PowerShell di Exchange Online.](https://go.microsoft.com/fwlink/p/?LinkId=396554)
  
> [!NOTE]
> I cmdlet per la creazione di report per altri servizi, come SharePoint Online, Skype for Business online e l'attività degli utenti di Microsoft 365, sono disponibili in PowerShell di Exchange Online. Per ulteriori informazioni, vedere [Cmdlet per i report in Exchange Online.](https://go.microsoft.com/fwlink/p/?LinkId=691595). 
  
## <a name="sharepoint-online-powershell-cmdlets"></a>Cmdlet di PowerShell per SharePoint Online

Per i cmdlet di SharePoint Online, vedere [Index of Windows PowerShell for SharePoint Online cmdlets.](https://go.microsoft.com/fwlink/p/?LinkId=691476)
  
Per istruzioni sulla connessione per PowerShell di SharePoint Online, vedere [Configurare SharePoint Online Management Shell Windows PowerShell ambiente.](https://go.microsoft.com/fwlink/p/?LinkId=691603)
  
## <a name="skype-for-business-online-powershell-cmdlets"></a>Cmdlet di PowerShell per Skype for Business Online

Per gli argomenti di riferimento sui cmdlet di Skype for Business online, vedere [Cmdlet di Skype for Business online.](https://technet.microsoft.com/library/mt228132.aspx)
  
Per istruzioni sulla connessione per PowerShell per Skype for Business online, vedere [Gestire Skype for Business online con PowerShell.](manage-skype-for-business-online-with-microsoft-365-powershell.md)

## <a name="security--compliance-center-powershell-cmdlets"></a>Cmdlet & PowerShell per Centro sicurezza e conformità

I riferimenti ai cmdlet del Centro sicurezza & conformità sono disponibili nella sezione di riferimento della documentazione [di PowerShell & sicurezza e conformità.](https://docs.microsoft.com/powershell/exchange/scc-powershell)
  
Per istruzioni sulla connessione per PowerShell & Centro sicurezza e conformità, vedere Connettersi a PowerShell per Centro [sicurezza & conformità.](https://docs.microsoft.com/powershell/exchange/connect-to-scc-powershell)
