# <a name="disable-access-to-sway-with-powershell-for-microsoft-365"></a>Disabilitare l'accesso a Sway con PowerShell per Microsoft 365

*Questo articolo si applica sia a Microsoft 365 Enterprise che a Office 365 Enterprise*.

Lo script ManageSway.ps1 PowerShell consente di visualizzare e disabilitare i servizi nell'organizzazione di Microsoft 365, incluso Sway. Questo script consente di automatizzare le procedure descritte negli argomenti seguenti:
  
- [Visualizzare licenze e servizi con PowerShell](view-licenses-and-services-with-microsoft-365-powershell.md)
    
- [Disabilitare l'accesso ai servizi con PowerShell](disable-access-to-services-with-microsoft-365-powershell.md)
