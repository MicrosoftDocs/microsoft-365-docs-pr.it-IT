# <a name="plan-for-security-amp-compliance"></a>Pianificare la conformità della &amp; sicurezza

La gestione della sicurezza e della conformità è una partnership. L'utente è responsabile della protezione di dati, identità e dispositivi, mentre Microsoft protegge con vigore i servizi di Microsoft 365. È possibile usare Insieme Microsoft 365 ed Enterprise Mobility + Security (EMS) per ottenere il livello di protezione appropriato per l'organizzazione.
  
## <a name="step-1-review-capabilities"></a>Passaggio 1: esaminare le funzionalità

Orientarsi alle funzionalità di protezione delle informazioni nel poster di Information Protection per Office 365. 
  
[Distribuire la protezione delle informazioni per le normative sulla privacy dei dati con Microsoft 365](../solutions/information-protection-deploy.md?view=o365-worldwide)
  
## <a name="step-2-check-your-secure-score"></a>Passaggio 2: controllare il punteggio sicuro

Dopo aver impostato l'abbonamento a Microsoft 365, prendere nota del punteggio iniziale. Secure Score fornisce suggerimenti di configurazione che puoi adottare per aumentare il punteggio. L'obiettivo è tenere presenti le opportunità che è possibile adottare per proteggere l'ambiente, senza influire negativamente sulla produttività degli utenti.
  
- [Introduzione al punteggio sicuro di Office 365](../security/mtp/microsoft-secure-score.md)
    
## <a name="step-3-plan-access-protection-for-identity-and-devices"></a>Passaggio 3: Pianificare la protezione dell'accesso per identità e dispositivi

Proteggere l'accesso ai dati e ai servizi di Microsoft 365 è fondamentale per difendersi da attacchi informatici e proteggersi dalla perdita di dati.
