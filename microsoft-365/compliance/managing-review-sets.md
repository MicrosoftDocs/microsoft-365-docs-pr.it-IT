# <a name="manage-review-sets-in-advanced-ediscovery"></a>Gestire i set di revisioni in Advanced eDiscovery

Gli insiemi da rivedere sono insiemi statici di documenti in cui è possibile analizzare, eseguire query, visualizzare, contrassegnare ed esportare i dati in un caso. Per ulteriori informazioni sull'esecuzione di queste attività, vedere:

- [Analizzare i dati in un set di recensioni](analyzing-data-in-review-set.md)

- [Eseguire query sui dati in un insieme da rivedere](review-set-search.md)

- [Visualizzare i documenti in un insieme da rivedere](view-documents-in-review-set.md)

- [Contrassegnare i documenti in un insieme da rivedere](tagging-documents.md)

- [Esportare i dati del caso](exporting-data-ediscover20.md)

## <a name="create-a-review-set"></a>Creare un set di recensioni

I set di revisione possono essere creati nella **scheda Set di revisioni** facendo **clic su + Aggiungi set di recensioni.**

![Aggiungere un set di recensioni](../media/f45c51d9-585d-47d1-b7fb-0288715e0b6a.png)

Nella pagina **a comparsa Aggiungi set di** revisione digitare un nome per il set di recensioni e quindi fare clic su **Salva.** Il nuovo set di revisioni viene visualizzato nell'elenco nella **scheda Set di** revisioni.

![Nuovo set di recensioni elencato nella scheda Set di recensioni](../media/AeDnewreviewset.png)

Esistono tre modi diversi per aggiungere dati a un set di recensioni in un Advanced eDiscovery caso.

![Tre modi per aggiungere un set di recensioni](../media/1f1f4efd-c03b-4255-bc3d-df358e56549c.png)

1. [Aggiungere i risultati della ricerca a un insieme da rivedere](add-data-to-review-set.md)

2. [Caricare dati non di Microsoft 365 in un insieme da rivedere](load-non-Office-365-data-into-a-review-set.md)

3. [Aggiunta di dati a un insieme da rivedere da un altro insieme da rivedere](add-data-to-review-set-from-another-review-set.md)
