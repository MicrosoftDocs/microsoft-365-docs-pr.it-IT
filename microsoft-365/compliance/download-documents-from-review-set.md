# <a name="download-selected-documents-from-a-review-set"></a>Scaricare i documenti selezionati da un set di revisioni

Il download offre un modo semplice per scaricare contenuto da un set di recensioni in formato nativo. Lo strumento di download in Advanced eDiscovery sfrutta le funzionalità di trasferimento dei dati del browser. Quando è pronto un download, verrà visualizzato un prompt del browser. I file scaricati con questo metodo vengono compressi in un file contenitore e conterranno file a livello di elemento. Ciò significa che se si sceglie di scaricare un allegato, si riceverà il messaggio di posta elettronica con l'allegato incluso. Analogamente, se si esporta un foglio Excel incorporato in un documento di Word, il documento di Word con il foglio Excel incorporato viene incluso nel download. Quando si scaricano elementi, la proprietà Last Modified Data viene mantenuta e può essere visualizzata come proprietà del file.

Per scaricare contenuto da un set di recensioni in un caso Advanced eDiscovery, iniziare selezionando i file che si desidera scaricare e quindi selezionare **Elementi azione**  >  **Scarica**.
