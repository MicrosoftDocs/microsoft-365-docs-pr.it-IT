# <a name="download-selected-documents-from-a-review-set"></a>Scaricare i documenti selezionati da un set di Revisione

Download offre un modo semplice per scaricare contenuto da un set di recensioni in formato nativo. Lo strumento di download in Advanced eDiscovery utilizza le funzionalità di trasferimento dei dati del browser. Quando il download è pronto, verrà visualizzato un prompt del browser. I file scaricati con questo metodo sono zippati in un file contenitore e conterranno file a livello di elemento. Questo significa che se si sceglie di scaricare un allegato, verrà visualizzato il messaggio di posta elettronica con l'allegato incluso. Analogamente, se si esporta un foglio di calcolo di Excel incorporato in un documento di Word, il documento di Word con il foglio di calcolo di Excel incorporato è incluso nel download. Quando si scaricano gli elementi, la proprietà Last modified data viene conservata e può essere visualizzata come una proprietà file.

Per scaricare contenuto da un set di revisione in un caso di eDiscovery avanzato, iniziare selezionando i file che si desidera scaricare, quindi selezionare il download **degli elementi di azione**  >  **Download**.
