# <a name="sharepoint-compatible-pdf-readers-that-support-microsoft-information-rights-management-services"></a>SharePoint-Compatible lettori PDF che supportano i servizi Microsoft Information Rights Management

Microsoft SharePoint 2013 supporta la protezione IRM (Information Rights Management) dei documenti PDF. Grazie a tale supporto, gli utenti possono caricare documenti PDF in raccolte protette con IRM e, al momento del download, i file verranno protetti tramite Microsoft Office IRM.
  
Per utilizzare i file PDF nelle raccolte protette dal proprietario con IRM, l'utente dovrà ottenere uno dei seguenti lettori compatibili con PDF:
