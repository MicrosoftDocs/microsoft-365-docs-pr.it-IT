# <a name="export-case-data-in-advanced-ediscovery"></a>Esportare i dati del caso in Advanced eDiscovery

Esistono tre modi per esportare i dati da un insieme da rivedere:

**Download:** Scarica (usando un browser) un piccolo set di file nativi. Questo è il modo più rapido per esportare un piccolo set di dati. Questo metodo mantiene i nomi dei file nativi. Per ulteriori informazioni, vedere [Scaricare documenti da un insieme da rivedere.](download-documents-from-review-set.md)

**Esportare:** Personalizzare i dati da esportare. Ciò include l'esportazione di metadati di file, file nativi, file di testo e documenti redatti che sono stati salvati in un file PDF. Dopo aver preparato i dati esportati, è possibile scaricarli dalla **scheda Esporta** in un computer locale. Per ulteriori informazioni, vedere [Esportare documenti da un insieme da rivedere](export-documents-from-review-set.md)

**Aggiungi a un altro insieme da rivedere:** Copiare i dati da un insieme di recensioni a un altro insieme di recensioni. Per ulteriori informazioni, vedere [Aggiungere dati da un insieme di recensioni a un altro insieme da rivedere.](add-data-to-review-set-from-another-review-set.md)

> [!NOTE]
> In Microsoft 365, viene eseguito l'hashing dei dati e tali hash vengono forniti nel file di output a scopo di verifica. Questa funzionalità è integrata dai log di controllo e dalla funzionalità di creazione di report, ad esempio statistiche di raccolta, rapporti sul set di carico e report di esportazione (incluso il file di caricamento dell'esportazione).
