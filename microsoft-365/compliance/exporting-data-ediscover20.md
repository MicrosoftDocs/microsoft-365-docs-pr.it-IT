# <a name="export-case-data-in-advanced-ediscovery"></a>Esportare i dati del caso in Advanced eDiscovery

Esistono tre modi per esportare i dati da un set di recensioni:

**Download:** Scarica (usando un browser) un piccolo set di file nativi. Questo è il modo più rapido per esportare un piccolo set di dati. Questo metodo mantiene i nomi di file nativi. Per ulteriori informazioni, vedere [Scaricare documenti da un set di revisioni.](download-documents-from-review-set.md)

**Esporta:** Personalizzare i dati esportati. Questo include l'esportazione di metadati di file, file nativi, file di testo e documenti revisionati già salvati in un file PDF. Dopo aver preparato i dati esportati, è possibile scaricarli dalla **scheda Esporta** in un computer locale. Per ulteriori informazioni, vedere [Export documents from a review set](export-documents-from-review-set.md)

**Aggiungi a un altro set di recensioni:** Copiare i dati da un set di recensioni a un set di revisione diverso. Per ulteriori informazioni, vedere [Aggiungere dati da un set di recensioni a un altro set di recensioni.](add-data-to-review-set-from-another-review-set.md)

> [!NOTE]
> In Microsoft 365, viene eseguito l'hashing dei dati e tali hash vengono forniti nel file di output a scopo di verifica. Questa funzionalità è integrata dai log di controllo e dalle funzionalità di creazione di report, ad esempio statistiche di raccolta, rapporti sul set di carico e report di esportazione (incluso il file di caricamento dell'esportazione).
