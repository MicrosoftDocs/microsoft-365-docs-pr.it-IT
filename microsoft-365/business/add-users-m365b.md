## <a name="add-new-users"></a>Aggiungere nuovi utenti

Nell'interfaccia di amministrazione passare alla scheda **Utenti** \> **Aggiungi utente**.
  
![Choose Add a user on the Users card in the admin center](media/55218f5b-899c-41cb-8486-8746fcef1748.png)
  
Nel riquadro **Nuovo utente** digitare le informazioni richieste.
