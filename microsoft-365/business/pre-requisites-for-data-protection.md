# <a name="prerequisites-for-protecting-data-on-devices-with-microsoft-365-for-business"></a>Prerequisiti per la protezione dei dati nei dispositivi con Microsoft 365 per le aziende

Questo articolo si applica a Microsoft 365 Business Premium.

Il primo passaggio per configurare l'organizzazione con Microsoft 365 per le aziende consiste nel verificare che sia possibile soddisfare i prerequisiti.
  
## <a name="requirements-for-setting-up-your-organization-with-microsoft-365-for-business"></a>Requisiti per la configurazione dell'organizzazione con Microsoft 365 per le aziende

- Windows dispositivi devono essere in esecuzione Windows 7 Professional, Windows 8 Pro o Windows 8.1 Pro.
    
    [Eseguire l'aggiornamento di dispositivi Windows a Windows Pro Creators Update](upgrade-to-windows-pro-creators-update.md) .
    
    Se stai eseguendo Windows 10 Home, devi **acquistare** Windows 10 Pro. Per [istruzioni, vedere upgrade Windows 10 Home to Windows 10 Pro.](../business-video/upgrade.md) 
    
- Rimuovere i dispositivi dalle soluzioni di gestione dei dispositivi mobili (Mobile Iron, AirWatch e cos√¨ via). Verranno registrate tutte le persone dell'organizzazione in Microsoft 365 per la gestione dei dispositivi mobili aziendali.
    
- Apple iOS 8.0 e versioni successive.
    
    Google Android 4.0 e versioni successive (incluso Samsung KNOX Standard 4.0 e versioni successive). Per altre informazioni, vedi [Dispositivi supportati da Intune.](/mem/intune/fundamentals/supported-devices-browsers)
    
- Se si dispone di applicazioni Office esistenti nei computer utente, leggere Preparare l'installazione [client di Office](prepare-for-office-client-deployment.md) per comprendere i passaggi che potrebbe essere necessario eseguire prima di poter configurare Microsoft 365 per le aziende per installare Office 2016 nei computer degli utenti.
