# <a name="prerequisites-for-protecting-data-on-devices-with-microsoft-365-for-business"></a>Prerequisiti per la protezione dei dati nei dispositivi con Microsoft 365 per le aziende

Questo articolo si applica a Microsoft 365 Business Premium.

Il primo passaggio per configurare l'organizzazione con Microsoft 365 per le aziende consiste nel verificare che sia possibile soddisfare i prerequisiti.
  
## <a name="requirements-for-setting-up-your-organization-with-microsoft-365-for-business"></a>Requisiti per la configurazione dell'organizzazione con Microsoft 365 per le aziende

- I dispositivi Windows devono eseguire Windows 7 Professional, Windows 8 Pro o Windows 8.1 Pro.
    
    [Eseguire l'aggiornamento di dispositivi Windows a Windows Pro Creators Update](upgrade-to-windows-pro-creators-update.md) .
    
    Se stai eseguendo Windows 10 Home, devi **acquistare** Windows 10 Pro. Per istruzioni, vedi aggiornare [Windows 10 Home a Windows 10 Pro.](https://support.microsoft.com/office/0aee10c1-4d34-43ee-a325-579c6c2df90e) 
    
- Rimuovere i dispositivi dalle soluzioni di gestione dei dispositivi mobili (Mobile Iron, AirWatch e cos√¨ via). Tutte le persone dell'organizzazione verranno registrate nella gestione dei dispositivi mobili di Microsoft 365 per le aziende.
    
- Apple iOS 8.0 e versioni successive.
    
    Google Android 4.0 e versioni successive (incluso Samsung KNOX Standard 4.0 e versioni successive). Per ulteriori informazioni, vedere Dispositivi [supportati da Intune.](https://go.microsoft.com/fwlink/p/?linkid=852307)
    
- Se si dispone di applicazioni di Office esistenti nei computer degli utenti, leggere preparare l'installazione client di [Office](prepare-for-office-client-deployment.md) per comprendere i passaggi che potrebbe essere necessario eseguire prima di configurare Microsoft 365 per le aziende per installare Office 2016 nei computer degli utenti.
