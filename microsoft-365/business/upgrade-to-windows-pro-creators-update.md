## <a name="upgrade-to-windows-10-pro"></a>Eseguire l'aggiornamento a Windows 10 Pro
  
Per eseguire l'aggiornamento a Windows 10 Pro, sono disponibili diverse opzioni. È possibile:
    
- Installare l'aggiornamento dal sito Download [software Microsoft](https://go.microsoft.com/fwlink/?LinkID=836951 ) Selezionare questa opzione se il dispositivo connesso si trova nello stesso dispositivo di quello che si &ndash; desidera aggiornare. 

    - Dal sito di download del software, fai clic su Aggiorna **ora** per avviare l'aggiornamento del dispositivo a Windows 10 Pro Creators Update. 
    
- Crea un supporto [](https://go.microsoft.com/fwlink/?LinkID=836960) di installazione con lo strumento per la creazione di supporti Selezionare questa opzione per creare un supporto di installazione di &ndash; Windows 10 Pro Creators Update (unità flash USB o file ISO) per installare Windows 10 in un PC diverso da quello in uso.

    - Leggere le istruzioni su come usare lo strumento e creare il supporto di installazione. 

> [!NOTE]
> Se hai dispositivi Windows che eseguono Windows 7 Pro, Windows 8 Pro o Windows 8.1 Pro, l'abbonamento a Microsoft 365 per le aziende ti autorizza a un aggiornamento a Windows Pro 10.
    
Vedi [Configurare i dispositivi Windows per gli utenti di Microsoft 365 per](set-up-windows-devices.md) le aziende per completare la configurazione dei dispositivi Windows 10. 
  
Vedi [Configurare i dispositivi mobili per gli utenti di Microsoft 365 per](set-up-mobile-devices.md) le aziende per completare la configurazione dei dispositivi Android e iOS.
