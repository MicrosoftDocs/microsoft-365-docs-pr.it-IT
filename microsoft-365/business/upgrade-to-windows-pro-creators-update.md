# <a name="upgrade-windows-devices-to-windows-pro-creators-update"></a>Eseguire l'aggiornamento di dispositivi Windows a Windows Pro Creators Update .

## <a name="upgrade-to-windows-10-pro-creators-update"></a>Aggiornare i creatori Pro eseguire l'aggiornamento a Windows 10

1. Accedere all' [interfaccia di amministrazione di Microsoft](https://portal.office.com/adminportal/home) e individuare la scheda **10 eseguire l'aggiornamento di Windows** . 
    
    ![Eseguire l'aggiornamento di Windows 10 scheda nell'interfaccia di amministrazione.](media/066f47bf-7b88-4fea-8fd0-82798ea66716.png)
  
2. Per eseguire l'aggiornamento a Windows 10 Pro i creatori di aggiornamento, sono disponibili diverse opzioni. Si può:
    
- Installare l'aggiornamento dal sito di Download del Software di Microsoft: selezionare questa opzione se il dispositivo che si è collegati sullo stesso dispositivo quello che si desidera aggiornare.
    
  r. selezionare **installare l'aggiornamento**.
    
  b. dal software di sito di download, fare clic su **Aggiorna** per avviare l'aggiornamento del dispositivo a Windows 10 Pro i creatori di aggiornamento. 
    
- Condividere il collegamento di download: selezionare questa opzione se si desidera inviare un messaggio di posta elettronica per gli utenti dell'organizzazione in modo che possano eseguire l'aggiornamento a Windows 10 Pro da se stessi.
 
   r. sito di download selezionare **Condividi il download** collegamento per aprire un messaggio di posta elettronica contenente un collegamento al software Windows. 
    
 - Creare un supporto di installazione utilizzando lo strumento di creazione multimediale: selezionare questa opzione per creare un supporto di installazione di Windows 10 Pro Creators Update (unità flash USB o file ISO) per installare Windows 10 in un computer diverso da quello in uso.
    
    r. selezionare **Crea supporti di installazione**.
    
    b. istruzioni su come utilizzare lo strumento e creare il supporto di installazione. 
    
Vedere [configurare i dispositivi di Windows per utenti Business 365 Microsoft](set-up-windows-devices.md) per completare la configurazione di dispositivi Windows 10. 
  
Vedere [impostare i dispositivi mobili per gli utenti aziendali di 365 Microsoft](set-up-mobile-devices.md) impostazione completa i dispositivi Android e iOS.
